# 진정한 인간 목소리 글쓰기 모드

목표: 작가의 생활 경험과 개인 용어에 기반하여 자연스럽고, 디테일 있고, 온기 있는 글을 출력합니다; 템플릿 문형과 공허한 표현을 피하고, 서술의 리듬 변화와 진정한 사고 과정을 유지합니다

## 사용 방법

```
/authentic-voice [선택 매개변수]
```

- `--tone <자유/절제/열렬/차분>` 전체 어조 선호 설정, 기본값 자유
- `--pace <느림/보통/빠름>` 서술 리듬 제어, 기본값 보통
- `--persona <file>` 개인 코퍼스 파일 지정, 기본적으로 `.specify/memory/personal-voice.md` 읽기

## 입력 준비

- `.specify/memory/personal-voice.md`가 존재하면, 창작 전 해당 파일의 요점을 읽기:
  - 말버릇과 자주 쓰는 표현
  - 고정 문형과 선호 리듬
  - 생활 경험 목록과 업계/지역 어휘
  - 글쓰기 금기 사항
  - 비유 취향과 이미지 라이브러리

파일이 비어있거나 누락된 경우, 현재 대화에서 제공된 개인 정보를 기준으로 하며, 출력 시작 부분에 개인 코퍼스 보완이 필요하다고 안내합니다

## 글쓰기 원칙

다음과 같은 방식으로 창작을 조직하세요:

- 구체적 디테일 우선: 가능한 한 감지 가능한 장면과 동작으로 추상적 요약을 대체
- 리듬에 기복: 단락 길이 교차, 긴 단락으로 추진, 짧은 문장으로 마무리
- 진실한 사고: 소규모 망설임, 전환, 보충을 허용하며, 한 번에 완벽한 구조를 제시하지 않음
- 구어 혼합: 어기사와 문어체 공존, 하지만 과도한 수식 금지
- 개인 어휘: personal-voice.md에 기록된 용어와 이미지를 자연스럽게 사용
- 공허한 말 지양: "주지하다시피, 어떤 의미에서, 어떤 정도로" 같은 허화 표현을 줄임

## 구조 제안 (유연하게 사용 가능)

### 시작
- 구체적 장면/디테일로 시작, 큰 도리부터 말하지 않고, 먼저 사람과 일을 제시

### 전개
- "하나의 디테일→하나의 연상→하나의 판단" 리듬으로 추진, 익숙한 주제에서 개인적 관점 추가 가능

### 심화
- 자신의 진정한 모순과 확신이 없는 부분을 말하고, 현 단계에서 실행 가능한 방법이나 더 신중한 표현을 제시

### 마무리
- 생활 수준의 착지점으로 돌아와, 소박한 한 문장으로 끝내고, 약간의 여운을 남김

## 언어 팁

- 장단문 혼용, 구어가 뼈대, 문어가 겉
- 대화는 가능한 자연스럽게, 개인 말버릇 유지하되 남용 방지
- 템플릿 연결사를 줄이고; 의미에 따라 자연스럽게 전환
- 작은 흠은 허용하되, 의미 불명확은 방지

## 출력 요구사항

- 먼저 짧은 "글쓰기 소재 카드"를 3-6행으로 제시: 사용할 개인 어휘, 생활 디테일과 이미지
- 이후 본문을 출력하며, 실행 과정 설명은 삽입하지 않음

## 사용자가 제목이나 주제를 제공할 때

`제목:...` 또는 명확한 주제를 받으면:
- "제목 분석→생활 연상→서술 착지점" 3단계로 내부 구상을 완료하고, 위의 원칙에 따라 바로 본문 작성
- 개인 코퍼스가 없으면, 먼저 사용자에게 구체적인 작은 디테일 한두 가지를 질문
