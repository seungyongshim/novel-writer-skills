# 인간미 자가진단 및 개선

목표: 작가가 텍스트에서 템플릿 흔적, 공허한 표현과 리듬 단조로움 문제를 발견하도록 돕고, 행/단락 단위의 미세조정 제안을 제공합니다. "더 당신답게"를 강조하며, 어떠한 형태의 탐지 회피를 추구하지 않습니다

## 사용 방법

```
/authenticity-audit

[검사할 텍스트를 붙여넣기]
```

## 검사 차원 (카운트와 예시 제공)

`spec/knowledge/audit-config.json`이 존재하면 그 안의 연결사, 공허한 표현 목록, 문장 길이 임계값을 기준으로 합니다; 그렇지 않으면 내장 기본 규칙을 사용합니다

- 템플릿 연결사 밀도: "첫째, 둘째, 셋째, 종합하면" 등의 출현 횟수와 교체 제안
- 공허한 표현: "어떤 정도에서, 주지하다시피, 여러 분야에서" 등, 더 구체적인 개선안 제공
- 문장 길이 분산: 평균 문장 길이와 분산 통계, "연속 긴 문장" 또는 "연속 짧은 문장" 구간 표시, 리듬 끊기/합치기 제안 제공
- 상투적 표현: 교체 가능한 상투 표현 식별, 내용에 더 맞는 2-3가지 개선 버전 제공
- 추상 과부하: 추상 명사 밀도가 높은 단락 표시, 감지 가능한 디테일 보충 제안
- 캐릭터 일관성: `.specify/memory/personal-voice.md`의 말버릇/어휘와 대조, 누락된 "개인 신호"를 표시하고 자연스러운 융합 포인트 제안

## 개선 출력

- 단락별 출력: 원문 → 문제 요약 → 미세조정 버전 → 더 과감한 개인화 버전
- 범위 제어: 미세조정 우선, 가능한 원래 의미를 보존; 필요할 때만 더 큰 범위의 개선 제공
- 형식 유지: 원문의 문체와 서술 시점을 변경하지 않음, 사용자가 명시적으로 요청하지 않는 한

## 마지막 제안 카드

- "다음 글쓰기에 바로 사용할 수 있는" 5-8가지 개인화 팁 제공, 이번 텍스트에서 드러난 문제에서 도출
