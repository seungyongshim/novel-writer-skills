# Authentic Voice 플러그인

글이 더 당신답게 쓰여지도록: 생활 디테일, 구어 리듬과 개인 어휘 일관성을 강조합니다. 이 플러그인은 "독창성과 자연스러움 향상"에 집중하며, 어떠한 탐지 회피 시나리오에도 관여하지 않습니다.

## 설치

```
novel plugins add authentic-voice
```

## 개인 코퍼스 준비

`.specify/memory/personal-voice.md`에 다음을 작성하세요:
- 말버릇과 자주 쓰는 표현
- 고정 문형과 리듬 선호
- 생활 경험 목록과 업계/지역 어휘
- 글쓰기 금기 사항
- 비유 취향과 이미지 라이브러리

## 주요 명령

- `/authentic-voice` 진정한 인간 목소리 글쓰기를 시작하고, 개인 코퍼스를 자동으로 읽어 출력 전 "소재 카드"를 제공합니다
- `/authenticity-audit` 기존 텍스트에 대해 인간미 자가진단 및 개선 제안을 합니다
- `/expert authentic-editor` 진정한 인간 목소리 편집 전문가를 호출하여 더 정밀한 행 단위 개선을 진행합니다

## 적용 시나리오

- 정식 창작: 안정적인 개인 스타일과 리듬이 필요할 때
- 2차 다듬기: 공허한 표현, 템플릿화, 리듬 단조로움 방지
- 장기 프로젝트: 점진적으로 개인 어휘와 이미지를 축적하여 안정적인 "작가 지문" 형성
