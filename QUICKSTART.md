# Novel Writer Skills 빠른 시작

## 5분 만에 시작하기

### 1. 설치 (1분)

```bash
npm install -g novel-writer-skills
```

### 2. 프로젝트 생성 (30초)

```bash
novelwrite init my-novel
cd my-novel
```

### 3. Claude Code에서 시작 (3분)

1. Claude Code에서 `my-novel` 폴더를 엽니다
2. 첫 번째 명령어를 입력합니다:

```
/constitution
```

3. AI의 안내를 따라 질문에 답합니다
4. 완료 후 계속 진행합니다:

```
/specify
/clarify
/plan
/tasks
/write
```

## 이게 전부입니다!

### 자동 활성화되는 Skills

아무것도 하지 않아도 다음 Skills가 자동으로 작동합니다:

- 🎨 **장르 지식**: "로맨스"/"미스터리"/"판타지" 언급 시 자동 활성화
- ✍️ **글쓰기 기법**: 대화나 장면 작성 시 자동 적용
- 🔍 **품질 검사**: 글쓰기 과정에서 백그라운드 모니터링

### 추천 첫날 일정

**소요 시간: 2-3시간**

```
09:00 - 09:20  /constitution  (창작 원칙)
09:20 - 10:00  /specify       (스토리 스펙)
10:00 - 10:15  /clarify       (모호한 점 명확화)
10:15 - 11:00  /plan          (창작 계획)
11:00 - 11:30  /tasks         (태스크 분해)
11:30 - 12:00  /write         (1장 집필 시작)
```

## 자주 묻는 첫 질문

**Q: 반드시 순서대로 실행해야 하나요?**
A: 순서대로 하는 것을 권장하지만, 최소 플로우는: `/constitution` → `/specify` → `/write`

**Q: Skills가 자동으로 활성화되나요?**
A: 네! 관련 내용(로맨스, 미스터리 등)을 언급하기만 하면 Skills가 자동으로 작동합니다

**Q: /plan을 건너뛸 수 있나요?**
A: 가능하지만, `/plan`의 안내가 있으면 `/write`가 더 효과적입니다
