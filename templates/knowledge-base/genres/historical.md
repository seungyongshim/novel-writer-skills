# 역사소설 창작 지식 베이스

## 빠른 참조 (Quick Reference)

역사소설의 핵심은 **시대 분위기와 역사적 진실의 균형**입니다. 당신은 허구의 이야기를 하고 있지만, 배경은 반드시 신빙성 있는 역사 세계여야 합니다.

**정의적 특징**:
1. **역사적 배경**: 이야기가 명확한 역사 시기에 발생 (보통 현재로부터 50년 이상)
2. **시대 분위기**: 복식, 언어, 사회 구조, 가치관이 시대에 부합
3. **진실과 허구의 균형**: 큰 역사는 진실, 작은 디테일은 허구 가능

**핵심 공식**:
```
역사소설 = 역사 고증 (30%) + 시대 분위기 (40%) + 이야기성 (30%)
```

**유형 분류**:
- **정사 지향**: 엄격한 고증, 사실 존중 (예: 《명나라 그 일들》)
- **극화 지향**: 일부 역사를 가공, 오락 위주 (예: 《보보경심》)
- **가공 역사**: 역사를 참고하되 완전히 허구 (예: 《낭야방》)

본 지식 베이스는 **역사소설** (고대 배경)의 창작에 초점을 맞춥니다.

---

## 핵심 원칙 (Core Principles)

### 1. 큰일은 허구하지 않고, 작은 일은 구애받지 않는다

**원칙**: 중대한 역사적 사건과 인물은 임의로 변경할 수 없고, 디테일은 합리적으로 허구할 수 있습니다.

**"큰일" (허구 불가)**:
- ✅ **중대한 역사적 사건**: 왕조 교체, 유명 전투, 역사적 전환점
  - 예시: "당나라에서 안사의 난이 일어나지 않았다"고 쓸 수 없음
- ✅ **유명 역사 인물의 기본 설정**: 생몰년, 주요 업적, 역사적 위치
  - 예시: "이백이 무장이었다"고 쓸 수 없음
- ✅ **시대 대배경**: 사회 제도, 과학기술 수준, 지리 상황
  - 예시: 송대에 총기가 등장할 수 없음

**"작은 일" (허구 가능)**:
- ✅ **허구의 인물**: 주인공, 조연
- ✅ **역사의 틈새 이야기**: 사서에 기록되지 않은 일상생활
- ✅ **합리적 디테일 보충**: 대화, 심리 활동, 장면 묘사
  - 사서에 "이세민이 대신을 접견했다"고 기록
  - 대화 내용을 허구할 수 있음 (인물 성격에 부합한다면)

**균형 기법**:

```
예시: 조선시대 배경 이야기 집필

【큰일 (반드시 존중)】
- 태조 이성계의 조선 건국 (1392년)
- 세종의 한글 창제 (1443년)
- 임진왜란 (1592년)

【작은 일 (허구 가능)】
- 가상의 의금부 도사 캐릭터
- 그와 어떤 궁녀의 사랑을 허구
- 사서에 기록되지 않은 작은 사건에 참여하는 것을 허구

【마지노선】
- 큰 역사 진행을 변경하지 않음
- 허구 인물이 실제 역사 인물의 업적을 대체하지 않음
- 허구 사건이 사실과 모순되지 않음
```

### 2. 역사 백묘법

**원칙**: 역사소설의 서술은 간결하고, 절제되어야 하며, 과도한 화려함을 피해야 합니다.

**왜 백묘를 사용하는가**:
- 고대 문헌의 기록 스타일에 부합
- 현대 웹소설의 "AI 투"를 회피
- 역사적 무게감을 더함

**백묘 기법**:

#### A. 사실을 서술하고, 수식하지 않는다
```
❌ 화려한 표현:
"영가의 난은 천지를 휩쓸어 버린 폭풍 같아, 천년 제국의 번영을 찢고,
수많은 생명을 절망의 심연으로 밀어 넣어, 피가 산하를 물들이고, 곡성이 들판에 가득했다."

✅ 백묘 표현:
"영가의 난 이후, 중원이 이민족에 점령되었다.
한족 사족과 백성 중 고향을 떠나기 싫은 소수를 제외하면, 대부분 남쪽으로 강을 건넜다."

(사서 기록처럼 간결)
```

#### B. 고문의 리듬감을 사용
```
【특징】
- 단문 위주 (10-20자)
- 명사, 동사를 많이 사용
- 형용사, 부사를 적게 사용
- 문형이 간결

예시:
"왕초가 이 몇 년간 유민 백여 명을 모아 자기 땅을 경작하게 했다."
(17자, 사실 서술, 불필요한 수식 없음)
```

#### C. 감정 묘사를 절제
```
❌ 과도한 감정:
"그의 마음은 칼로 베이는 것 같고, 눈물이 두 눈을 가려,
온 세상이 무너졌고, 절망이 조수처럼 그를 삼켰다."

✅ 절제된 표현:
"그는 한참 침묵하더니, 결국 두 글자만 말했다: '가자.'"
(감정을 행동으로 암시, 직접 쏟아내지 않음)
```

**백묘는 재미없음이 아니다**:
- ✅ 여전히 드라마틱 긴장감이 있을 수 있음 (플롯 추진으로)
- ✅ 여전히 디테일이 있을 수 있음 (하지만 의미 있는 디테일 선택)
- ✅ 여전히 감정이 있을 수 있음 (하지만 행동과 대화로 전달)

### 3. 시대 분위기 조성

**원칙**: 독자가 이것이 고대라고 "느끼"게 해야 하며, 현대인이 고대 의상을 입은 것이 아니어야 합니다.

**핵심 차원**:

#### A. 언어 스타일
```
【현대 어휘 금지】
❌ "이 방안은 매우 과학적이다"
❌ "우리는 팀워크가 필요하다"
❌ "기회를 한번 주세요"

✅ 고대식 표현으로 대체:
"이 방법이 이치에 매우 합당합니다"
"상하가 일심이 되어야, 일을 이룰 수 있습니다"
"아무쪼록 대인께서 살펴주소서"

【주의】
- 완전한 고전 문어체일 필요 없음 (독자가 힘듦)
- 구어체 위주 + 고전적 어구 곁들임
- 핵심 대화에 고전적 표현으로 분위기 강화
```

**현대 어휘 블랙리스트**:
| 현대 어휘 | 고대 대체 |
|--------|---------|
| 과학적 | 이치/도리 |
| 로맨틱 | 풍류/운치 |
| 프라이버시 | 사적/은밀 |
| 개인 | 자신/본인 |
| 사회 | 세상/천하 |
| 문제 | 난처/병통 |
| 기회 | 시기/계기 |
| 조건 | 자본/밑천 |

#### B. 사회 구조
```
【계급 차이】
고대 사회는 등급이 엄격하여, 반드시 체현해야:

양반 vs 상민 (조선)
귀족 vs 평민 (고려)
관 vs 민 (역대)

예시:
❌ "그가 현감과 대등하게 대화했다"
✅ "그가 바닥에 무릎을 꿇고, 현감은 상좌에 앉아, 담담히 한마디만 물었다"

【호칭 체계】
- 왕: 전하, 주상, 상감마마
- 관원: 대감, 나으리, 대인
- 평민: 소인, 초야지민
- 여성: 마마, 마님, 소녀

【주의】
시대마다 호칭이 다르니, 고증이 필요.
```

#### C. 일상생활
```
【의식주행】
- 의: 한복 형태 (관복, 평복, 예복)
- 식: 고대 식재료 (고추는 임진왜란 이후)
- 주: 건축 구조 (좌북조남, 전당후침)
- 행: 교통수단 (말, 가마, 도보)

【시간 표현】
❌ "오후 3시"
✅ "신시 삼각"

고대 12시진:
자(23-1), 축(1-3), 인(3-5), 묘(5-7)
진(7-9), 사(9-11), 오(11-13), 미(13-15)
신(15-17), 유(17-19), 술(19-21), 해(21-23)

【화폐】
시대마다 다름:
- 고려: 은병, 동전
- 조선: 냥, 전, 돈, 문

【주의】
은 1냥의 구매력이 시대에 따라 크게 다르니, 고증 필요.
```

#### D. 가치관
```
【고대 사고방식】
- 충군: 왕의 권위를 도전할 수 없음
- 효도: 부모의 명은 어길 수 없음
- 종족: 가문의 이익이 개인보다 우선
- 남존여비: 여성 지위가 낮음 (객관적으로 보여주되, 미화할 필요 없음)

【예시】
❌ 현대적 사고:
"내 인생은 내가 결정해, 부모님이 뭐라 하든 상관없어!"

✅ 고대적 사고:
"부모님의 명이니, 감히 어기지 못합니다. 다만 이 일은 가문의 존망이 걸린 것이니,
아버님께서 삼가 다시 생각해 주시옵소서."
(반항하더라도, 고대의 논리에 맞아야)

【주의】
- 전통을 돌파하는 캐릭터가 있을 수 있음 (드라마성)
- 하지만 돌파에는 대가가 따라야 (저항 없이는 불가)
- 다른 캐릭터가 고대의 정상적 반응을 보여야 (주인공과 대비)
```

### 4. 고증과 오락의 균형

**원칙**: 역사소설은 역사 교과서가 아닙니다. 이야기성이 우선이되, 터무니없어서는 안 됩니다.

**균형 기준**:

#### A. 핵심 설정은 반드시 고증
```
【반드시 정확해야 할 내용】
- 관직 체계
- 중대 역사 사건 타임라인
- 지리 위치 (도시, 산천)
- 시대 특징적 제도

【도구】
- 정사 기록
- 각 시대 백과
- 학술 논문
```

#### B. 디테일은 합리적으로 허구 가능
```
【허구 가능한 디테일】
- 어떤 관원의 일상생활
- 어떤 연회의 구체적 대화
- 어떤 허구 인물의 경험

【허구 원칙】
- 시대 배경에 부합
- 사실과 모순되지 않음
- 합리적 상상 (근거 있음)

예시:
사서 기록: "세종이 대신을 접견했다."
허구 가능:
- 연회 메뉴 (조선시대 식단에 부합)
- 연회 대화 (인물 성격에 부합)
- 연회 디테일 (조선시대 예법에 부합)
```

#### C. 극화의 마지노선
```
【허용되는 극화】
- 허구의 로맨스 라인 (큰 역사에 영향 안 줌)
- 허구의 작은 사건 (역사의 틈새 이야기)
- 허구의 조연 (시대 군상을 풍성하게)

【허용되지 않는 극화】
- 중대 역사 진행 변경
- 유명 인물의 역사적 위치 변경
- 시대 과학기술 수준 위반

예시:
✅ 허용: "이순신과 한산도 대첩" (사실)
  → 구체적 전투 장면 디테일 허구 (극화)

❌ 불허: "이순신이 총기를 발명했다" (과학기술 위반)
  → 역사적 신빙성을 파괴
```

**점검 공식**:
```
고증도 = 핵심 설정 정확도 × 디테일 합리도

핵심 오류 (잘못된 시대) × 디테일이 아무리 합리 = 0점
핵심 정확 × 디테일 엉뚱 (조선시대에 감자) = 저점
핵심 정확 × 디테일 합리 = 고점
```

### 5. 역사 인물의 처리

**원칙**: 실제 역사 인물이 등장할 수 있지만, 사실을 존중하고 과도하게 각색하면 안 됩니다.

**처리 전략**:

#### A. 조연급 역사 인물 (적절히 허구 가능)
```
【정의】
- 사서에 기록이 있지만 상세하지 않음
- 핵심 역사 인물이 아님
- 예시: 어떤 소관원, 어떤 장수의 부장

【허구 허용】
- 성격 디테일
- 일상생활
- 주인공과의 상호작용

【마지노선】
- 사서의 기본 기록을 위반하지 않음
- 그들의 역사적 역할을 변경하지 않음
```

#### B. 유명 역사 인물 (엄격히 존중)
```
【정의】
- 사서에 상세히 기록
- 역사 진행에 중대한 영향
- 예시: 세종대왕, 이순신, 태조 이성계

【처리 원칙】
1. 기본 성격이 사서 기록에 부합
2. 중대 결정이 역사에 부합
3. 디테일을 보충할 수 있지만, 이미지를 전복하면 안 됨

【예시】
사서 기록: 태조 이성계는 위화도 회군으로 권력을 장악, 새 왕조 건국
처리:
✅ 그와 어떤 허구 캐릭터의 대화를 허구 가능
✅ 그의 내적 갈등을 묘사 가능
❌ "인자하고 나약한 임금"으로 쓸 수 없음 (위치 전복)
```

#### C. 가장 안전한 전략: 허구 주인공 사용
```
【권장 방법】
- 주인공은 허구
- 역사 인물은 조연/배경
- 주인공이 역사를 목격하되, 역사를 바꾸지 않음

【장점】
- 창작 자유도가 높음
- 사실 위반 걱정 없음
- 독자 몰입감이 강함

【예시】
《낭야방》
- 주인공: 허구의 매장소
- 배경: 남북조 유사 (하지만 가공)
- 역사 인물: 이름 변경 (논란 회피)
```

## 실천 적용 (Practical Application)

### `/specify` 단계에서

**역사소설 규격 템플릿**:

```markdown
## 역사적 배경 설정

### 시대 선택
- **시대**: [조선 초기/고려 말기/가공-유사 조선]
- **구체적 연도**: [1420년경, 세종 즉위 초]
- **역사적 단계**: [성세/난세/왕조 교체]

### 역사 유형
- [ ] 정사 지향 (엄격한 고증)
```

## 흔한 함정 (Common Pitfalls)

### 1. 시대 혼란

**문제**: 장삼이사, 다른 시대의 것이 뒤섞임.

```
❌ 전형적 오류:
- 고려시대에 "의정부" (조선시대 기관)
- 조선 초기에 "은화" (주요 동전 사용)
- 세종 시대에 "고추장" (임진왜란 이후)

⚠️ 고위험 어휘:
- 승상 (조선에 없음)
- 특무 (현대어, "도사", "나장" 사용)
```

**회피 방법**:
- 시대별 어휘 목록 만들기
- 물품, 관직, 호칭을 분류 정리
- 집필 시 대조 점검

---

### 2. 현대적 사고 침투

**문제**: 고대인이 현대어를 하고, 현대적으로 생각함.

```
❌ 현대적 사고:
"이 정책은 인성적이지 못하다."
"우리는 자유 평등을 추구해야 한다."
"여성에게도 선택할 권리가 있어야 한다."

✅ 고대적 사고:
"이 영이 너무 가혹하니, 민심을 잃을까 두렵습니다."
"사람의 운명은 각기 다르니, 어찌 억지로 하겠습니까."
"여자란 본시 아버지를 따르고 남편을 따르는 것이거늘, 무슨 선택이 있겠습니까?"

【주의】
- 전통을 돌파하는 캐릭터가 있을 수 있음 (드라마성)
- 하지만 이것이 "이류"임을 인식해야 (다른 사람이 반대)
- 대가가 따라야 (이해받지 못함, 저항에 부딪힘)
```

### 3. 과도한 고증

**문제**: 소설이 역사 교과서가 됨.

```
❌ 과도한 고증:
"조선시대 관복 제도는 이러했다:
1품 관복에는 학 자수, 2품 관복에는 운학 자수,
3품 관복에는 공작 자수, 4품 관복에는..."
(연속 2페이지 관복 소개)

독자 반응: "나는 소설 읽으러 왔지, 역사 수업 들으러 온 게 아니야!"
```

**회피 방법**:
```
✅ 자연스럽게 녹임:
"그가 자기 관복을 내려다보니, 가슴의 공작이 아직 선명했다.
3품, 높지도 낮지도 않고, 딱 적당했다."

(디테일로 정보를 전달, 해설을 펼치지 않음)
```

**점검 기준**:
- 역사 지식 삽입 < 3줄/회
- 반드시 플롯에 봉사 (해설을 위한 해설은 안 됨)
- 독자가 전후 맥락으로 이해 가능 (추가 설명 불필요)

### 4. 역사 인물이 씬을 빼앗음

**문제**: 주인공이 역사 인물에 의해 변두리화됨.

```
❌ 잘못된 예시:
- 주인공은 허구인데, 80% 분량이 세종대왕을 쓰고 있음
- 모든 핵심 결정은 세종이 내림
- 주인공은 그냥 관객

독자 반응: "그럼 그냥 세종 전기를 쓰지, 주인공은 왜 있어?"
```

**올바른 방법**:
```
✅ 주인공이 핵심, 역사 인물은 배경
- 역사 인물 등장은 적지만 정확 (화룡점정)
- 주인공이 허구 플롯을 추진 (역사의 틈새 이야기)
- 역사 인물이 주인공에 영향을 주지만, 대체하지 않음

예시:
- 세종이 하는 것: 중대 역사 결정 (한글 창제)
- 주인공이 하는 것: 허구의 사건 조사, 개인 성장
- 교차: 주인공이 역사를 목격하되, 역사를 바꾸지 않음
```

### 5. 가공이 철저하지 않음

**문제**: 가공하려 하면서 역사도 빌리려 하여, 양쪽 다 아닌 상태.

```
❌ 어색한 가공:
"이것은 가공 역사소설이지만, 인물은 전부 실재:
태조, 세종, 정조가 모두 등장..."

독자 반응: "이거 도대체 어느 시대야? 엉망이야!"
```

**명확한 선택**:

| 유형 | 역사 진실도 | 창작 자유도 | 대표작 |
|------|-----------|-----------|--------|
| 정사 지향 | 90% | 10% | 사극 다큐 |
| 극화 지향 | 60% | 40% | 《궁》 류 |
| 유사 가공 | 30% | 70% | 《낭야방》 유사 |
| 완전 가공 | 0% | 100% | 완전 창작 세계 |

**제안**:
- 엄격한 역사를 하든 (정사 지향)
- 철저히 가공을 하든 (이름 변경)
- "어느 시대 유사하지만 완전히는 아닌" 것은 하지 마세요 (독자 혼란)

**유형**: 유사 가공 (남북조 참조, 하지만 이름 변경)

```
【역사 참조】
- 시대 배경: 남북조 유사
- 사회 구조: 문벌 사족, 왕권
- 복식 예법: 남조 참고

【가공 처리】
- 왕조 이름: 대량 (허구)
- 인물 이름: 전부 허구
- 지리: 일부 허구

【장점】
✓ 역사적 무게감이 있으면서도
✓ 사실의 속박을 받지 않고
✓ 독자에게 고증 압박 없음

【왜 성공했는가】
- 시대 분위기 도달 (언어, 복식, 예법)
- 역사 디테일에 얽매이지 않음 (가공이 논란 면제)
- 이야기 자체에 집중 (권모, 복수, 형제애)
```
