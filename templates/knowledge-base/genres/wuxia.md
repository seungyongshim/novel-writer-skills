# 무협 소설 창작 지식 베이스

## 빠른 참조 (Quick Reference)

무협 소설의 핵심은 **강호 세계와 협의 정신**입니다.

**정의적 특징**:
1. **강호 배경**: 독특한 규칙을 가진 무림 세계
2. **무공 체계**: 합리적이고 계층이 있는 무학 설정
3. **협의 정신**: 의를 행하고, 은원을 갚음

**핵심 공식**:
```
무협 소설 = 강호 세계관 + 무공 체계 + 협의 정신 + 은원 정수
```

---

## 핵심 원칙 (Core Principles)

### 1. 강호 규칙

**강호 ≠ 조정**: 자체적인 운영 규칙이 있습니다.

**강호의 계층**:
- **백도**: 명문 정파 (소림, 무당, 아미)
- **흑도**: 마교, 사파
- **회색 지대**: 독행협, 은세 고수

**강호 규율**:
- 무예로 교류
- 은원 분명
- 장유유서
- 무고한 자를 해치지 않음

---

### 2. 무공 체계

**계층 분명**:
1. 삼류 - 일반 강호인
2. 이류 - 문파 엘리트
3. 일류 - 장문인/장로급
4. 절정 - 무림 신화

**무학 유형**:
- 내공 (진기 수련)
- 외공 (초식 기술)
- 경공 (신법)
- 암기 (원거리)

**제약 원칙**:
- 무적의 무공은 없음
- 상생상극
- 천부 + 노력 + 기연

---

### 3. 협의 정신

**"협"이란 무엇인가**:
- 의를 행하고 약자를 돕는 것
- 은원을 분명히 갚는 것
- 약속을 목숨보다 중히 여기는 것

**유형**:
- 대협: 나라와 백성을 위해
- 유협: 불의를 보면 나서는 것
- 은협: 산림에 은거

## 실전 적용 (Practical Application)

### 무공 명명 기법

```
【의경파】항룡십팔장, 독고구검, 능파미보
【실용파】철사장, 금강불괴신공
【시적파】답설무흔, 비화적엽
```

### 강호 문파 설정

```
명문 정파:
- 소림 (불문 무학)
- 무당 (도가 무학)
- 아미 (여성 문파)

사파:
- 마교 (패도 무공)
- 오독교 (독 사용)
```

### 클래식 장면

```
- 절벽에서 떨어져 고수를 만남
- 무림대회에서 맹주 쟁탈
- 도룡도 의천검
- 사부의 원수를 갚음
- 신분의 비밀
```

---

## 흔한 함정

1. **무공 무한 인플레이션**: 오늘 천하제일을 이기고, 내일 또 더 강한 자가 등장
2. **협의 구호화**: "내가 의를 행하겠소!" (공허함)
3. **문파 유형화**: 소림은 전부 좋은 사람, 마교는 전부 나쁜 사람

## Commands와의 통합

```markdown
## 강호 설정
- 시대 배경: [명나라/청나라/가상]
- 주요 문파: [3-5개]
- 무림 지보: [도룡도/무공 비급]
- 핵심 모순: [정사 대립/문파 원수]
```

## 총결

무협 소설의 매력:
- 통쾌한 은원의 강호
- 칼빛 검영의 격투
- 대협의 정수

**기억하세요**: 무협이 쓰는 것은 무공만이 아니라, 강호 속 인정과 의리입니다! ⚔️
