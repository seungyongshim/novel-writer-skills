# 독점 없는 규범

## 핵심 이념

**올바른 가치관, 엄밀한 논리** - 독자를 편하게 하고, 불쾌감을 주지 않습니다.

---

## 흔한 독점 목록

### ❌ 절대 금지

#### 1. 주인공 지능 저하
```
❌ 함정인 줄 알면서 뛰어듦
❌ 결정적 순간에 기술을 잊음
❌ 무뇌 충동
```

#### 2. 억지 오해
```
❌ 한마디면 해결되는데 굳이 안 말함
❌ 엿듣기는 반만 듣기
❌ 해명 기회를 주지 않음
```

#### 3. 성모 전개
```
❌ 무한정 용서
❌ 적의 선화
❌ 적에게 자비롭고 아군에게 가혹
```

#### 4. 도구인간 조연
```
❌ 경험치 제공만을 위해 존재
❌ 배경 역할만 함
❌ 합리적 동기 없음
```

#### 5. 이중잣대
```
❌ 주인공은 되고 다른 사람은 안 됨
❌ 내로남불
```
