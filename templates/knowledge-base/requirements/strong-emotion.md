# 강한 감정 규범

## 핵심 이념

**감정 충만, 충격력 강함** - 독자에게 강렬한 감정 체험을 제공합니다.

---

## 핵심 요구

### 1. 감정 폭발 포인트

매 10-15장마다 최소 1개 감정 폭발 포인트:
- 분노 폭발
- 슬픔 결핍
- 기쁨 충만
- 공포 극치

### 2. 감정 복선

**갑자기 폭발할 수 없고, 복선이 필요합니다**:
```
억압 (3장) → 축적 (2장) → 폭발 (1장)
```

### 3. 감정 표현

#### ✅ 행동을 통해
```
분노: 물건을 부수고, 떨리고, 이를 악물고
슬픔: 소리 없이 울고, 웅크리고, 멍하니
기쁨: 크게 웃고, 뛰어오르고, 안아주고
```

#### ❌ 직접 설명
```
"그녀는 매우 화가 났다" → 부족
"그녀는 화가 나서 온몸이 떨렸다" → 더 나음
"그녀는 컵을 집어 벽에 던졌다" → 최선
```

## 감정 리듬

**파도식**:
```
클라이맥스 → 완만 → 클라이맥스 → 완만
```

**금지**:
```
계속 클라이맥스 (독자 마비)
계속 완만 (독자 지루)
```
