# AI 느낌 제거 규칙 v4.0 (급진판)

## 설계 이념

**가장 급진적인 AI 느낌 제거 방안** - 진실감 우선, 일부 문학성 희생.

진짜 작가의 집필 습관을 모방하여, AI 흔적을 철저히 소멸합니다.

---

## 경고

⚠️ 본 규칙을 사용하면 문장이 매우 "직접적"이고 "단단"해집니다
⚠️ 일부 문학적 미감을 희생할 수 있습니다
⚠️ AI 느낌에 극도로 민감한 독자층에 적합합니다

## 제2층: 금지 문형

### 1. "그러나"로 시작하는 전환문

❌ **완전 금지**:
```
그러나, 일이 그가 생각한 것처럼 간단하지 않았다.
```

✅ **재구성**:
```
근데 틀렸다.
일이 그렇게 간단하지 않았다.
생각지도 못했는데...
```

---

### 2. "모를 리 없다"식 신의 시점

❌ **완전 금지**:
```
모를 리 없다, 이 모든 것이 몰래 관찰당하고 있다는 것을.
```

✅ **재구성 방안**:
- 방안1: 시점을 전환하여 직접 묘사
- 방안2: "그가 모르는 것은..."
- 방안3: 완전 삭제, 후속에서 자연스럽게 공개

---

### 3. "~속에서"의 부사구 구조

❌ **완전 금지**:
```
달빛 아래, 그녀의 얼굴이 유난히 아름다웠다.
```

✅ **재구성**:
```
달빛이 그녀 얼굴에 비쳤다.
얼굴에 달빛이 있으니, 꽤 예뻤다.
```

---

### 4. "마치...처럼"의 비유

❌ **완전 금지**:
```
그의 속도가 마치 한 줄기 바람처럼 빨랐다.
```

✅ **재구성**:
```
속도가 엄청 빨랐다.
바람처럼 빨랐다. (꼭 비유해야 한다면 간소화)
```

---

### 5. "바로 이 순간"류 시간 지시

❌ **완전 금지**:
```
바로 이 순간, 그녀의 기분이 유난히 복잡했다.
```

✅ **재구성**:
```
기분이 복잡했다.
지금 마음이 뒤죽박죽이다.
```

---

### 6. "...이 그녀를..." 인과문

❌ **최대한 회피**:
```
따뜻한 촉감이 그녀의 생각을 차츰 평온하게 했다.
```

✅ **재구성**:
```
컵이 좀 뜨거웠다. 손이 따뜻해졌다.
기분이 천천히 가라앉았다.
```

## 제3층: 형용사 제한

### 엄격한 제한

**규칙**:
- 100자당 형용사 최대 3개
- 연속 2개 이상 형용사 금지
- 모든 장식적 형용사 삭제

### 예시 비교

❌ **형용사 과다 (7개)**:
```
가녀린 하얀 손가락이 부드럽게 매끄러운 컵벽을 어루만지며,
정교한 도자기 컵에서 은은한 향이 풍겼다.
```

✅ **미니멀 형용사 (0개)**:
```
그녀가 컵을 들고, 손가락으로 컵벽을 문질렀다.
컵은 도자기고, 약간 향이 났다.
```

## 제4층: 사자성어 전면 금지

### 규칙

**모든 사자성어 사용 완전 금지** (일상 구어 사자성어 포함)

### 사자성어 대체표

| ❌ 사자성어 | ✅ 일상어 |
|---------|---------|
| 감개무량 | 말로 다 못 함 / 복잡함 |
| 어찌할 바를 모르다 | 어떻게 해야 할지 모르겠다 |
| 속수무책 | 반응을 못 했다 |
| 영문을 모르다 | 이해가 안 된다 |
| 반신반의 | 잘 안 믿겼다 |
| 아무렇지 않은 척 | 태연한 척 |
| 당연하다 | 원래 그래야 하는 것 |
| 서로 쳐다보다 | 너 보고, 나 보고 |
| 놀라서 입이 벌어지다 | 멍해졌다 |
| 할 말을 잃다 | 말이 안 나왔다 |
| 묵시적 양해 | 아무도 말 안 했지만 다 알았다 |

## 제5층: 대화 AI 제거

### 대화 태그 규칙

❌ **금지 태그**:
```
...하게 말했다, 대답했다, 추궁했다
걱정스럽게 물었다, 씁쓸히 웃으며 말했다
```

✅ **허용 태그**:
```
말했다, 물었다, 답했다 (극소 사용)
또는 태그 완전 생략
```

### 대화 묘사 규칙

❌ **금지 묘사**:
```
"어떻게 이럴 수 있어?" 그녀의 목소리에 약간의 떨림이 섞여,
눈시울이 살짝 붉어지며, 분명히 적지 않게 상처받은 모양이었다.
```

✅ **간소화 묘사**:
```
"어떻게 이럴 수 있어?" 목소리가 떨렸다.
"어떻게 이럴 수 있어?" 눈시울이 붉어졌다.
```

## 제6층: 단락 구조 제한

### 엄격한 규칙

1. **단락당 4문장 이하**
2. **대화 단락은 1문장만**
3. **심리 묘사 2문장 이하**

### 예시 비교

❌ **단락 과장 (7문장)**:
```
임완청이 카페에 앉아, 손에 라떼를 들고 있었다. 창밖을 보니
비가 점점 세지고 있었다. 오늘 아침 일이 떠올라 마음이 아직 좀 안 좋았다.
사실 별거 아닌데, 그냥 갑자기 지친 느낌. 컵을 들어 한 모금 마셨는데
커피가 이미 식어 있었다. 컵을 내려놓고 핸드폰을 꺼냈다가 다시 넣었다.
밖에서 우산 쓴 사람이 지나갔고, 한번 보다가 시선을 거뒀다.
```

✅ **단락 분할**:
```
임완청이 카페에 앉아, 라떼를 들고 있었다.
비가 점점 세졌다.

아침 일이 떠올랐다. 아직 좀 속상했다.

커피가 식었다. 컵을 내려놓고 핸드폰을 봤다가 다시 가방에 넣었다.

밖에서 우산 쓴 사람이 지나갔다. 한번 봤다.
```

## 제7층: 구두점 AI 제거

### AI의 흔한 구두점 습관 (금지)

❌ **금지 패턴**:
```
1. 말줄임표로 사고 표현: "그가 생각했다......"
2. 대시로 설명: "그녀가 웃었다 — 매우 복잡한 웃음이었다"
3. 과도한 느낌표: "정말 믿을 수 없어!!!"
```

✅ **진짜 사람 구두점**:
```
1. "그가 생각했다."
2. "그녀가 웃었다. 좀 복잡하게 웃었다."
3. "정말 믿을 수 없다."
```

## 극단적 규칙 (선택)

### 여전히 AI 느낌이 있다면, 다음 극단 규칙을 활성화:

#### 1. 문장 길이 하드 리밋
- 80% 문장 15자 이하
- 40자 초과 문장 금지
- 평균 문장 길이 12자 이내

#### 2. 단락 길이 하드 리밋
- 단락당 최대 3문장
- 대화 단락은 1문장만
- 5행 초과 단락 금지

#### 3. 모든 비유 금지
- "~같은", "마치", "~처럼" 사용 금지
- 모든 비유 기법 사용 금지
- 직접 묘사

#### 4. 모든 사자성어 금지
- 사자성어, 관용구 포함
- 전부 일상어로 분해
- 2음절 단어 위주

## 완전한 예시 비교

### 장면: 주인공이 위험을 만남

#### ❌ AI 느낌 강함 (전통 작법)

```
이명이 천천히 고개를 들었고, 깊은 눈동자에 진중한 기색이 스쳤다.
그는 명확하게 느낄 수 있었다, 주변 분위기가 이상하게 변했음을,
마치 무언가 나쁜 일이 곧 일어날 것 같았다.

그러나, 일은 그가 상상한 것보다 훨씬 복잡했다. 모를 리 없다,
이 겉보기에 평온한 숲 깊은 곳에서 살의에 가득 찬 한 쌍의 눈이
그의 일거수일투족을 관찰하고 있다는 것을. 바로 이 순간, 위기사방에
그는 전혀 알아차리지 못했다.
```

**문제 분석**:
- 금지어: 천천히, 깊은, 이상하게, 그러나, 모를 리 없다, 바로 이 순간
- 금지 문형: "마치...것 같다", "~속에서"
- 형용사 과다: 깊은, 진중한, 이상한, 평온한, 살의에 가득 찬
- AI화 표현: 명확하게 느낄 수 있었다, 일거수일투족, 전혀 알아차리지 못했다

---

#### ✅ v4.0 기준 (완전 AI 제거)

```
이명이 고개를 들었다.

주변이 너무 조용했다.

갑자기 뭔가 이상하다는 걸 깨달았다. 확 돌아봤다.

숲 속에서 뭔가 움직였다.

큰일이다.
```

**개선 분석**:
- 금지어 제로
- 사자성어 제로
- 비유 제로
- 극단 단문 (평균 5자)
- 한 문장 한 단락
- 거대한 여백

## 총결

**Anti-AI v4.0의 핵심**:

1. **무자비한 삭제** - 모든 AI 어휘 전부 삭제
2. **극단적 단문** - 문장은 짧을수록 좋다
3. **직백화** - 빙빙 돌리지 않는다
4. **장식 제거** - 모든 수식 삭제
5. **진짜 사람 느낌** - 진짜 사람의 말하기 습관 모방

---

**마지막 리마인더**:

> 이 버전은 극단적으로 절제적이며, AI 느낌에 제로 톨러런스인 작가에게 적합합니다.
> 너무 딱딱하다면, anti-ai-v3.md로 다운그레이드 사용 가능합니다.
