# 쾌감소설 집필 규범

## 핵심 이념

**리듬이 왕, 쾌감 우선** - 독자가 멈출 수 없게 합니다.

---

## 황금 법칙

### 법칙 1: 챕터당 최소 2개 쾌감 포인트

**쾌감 포인트 유형**:
1. 역전 (의심받음 → 실력 시현)
2. 과시 (저조함 → 경이적 표현)
3. 레벨업/돌파
4. 보물/스킬 획득
5. 반전 (열세 → 역전)
6. 상대 압도

### 법칙 2: 밀집된 갈등

**절대 허용 안 됨**:
- 연속 2장 갈등 없음
- 500자 초과의 일상 유수장
- 무의미한 환경 묘사
- 메인 라인을 지연하는 사이드 라인

### 법칙 3: 챕터마다 후크

**결미에 반드시 있어야**:
- 새로운 위기
- 예상치 못한 반전
- 강자 강림
- 신분 폭로
- 중요 정보

---

## 챕터 구조 템플릿

### 오프닝 (주의 끌기)

**3종 오프닝 방식**:

1. **갈등 오프닝**
```
"네가 뭔데, 나한테 말걸 자격이 있어?"

장소가 냉소했다.

이명이 고개를 들어 그를 봤다.
```

2. **서스펜스 오프닝**
```
이명이 문을 열었다.

방 안에, 있어서는 안 될 사람이 서 있었다.
```

3. **쾌감 오프닝**
```
"불가능해!"

장 장로가 눈을 크게 떴다.

이명 손안에서 금단이 천천히 회전하고 있었다.
```

---

### 중간 (플롯 추진)

**반드시 포함 (최소 2항)**:
- ✅ 1-2개 쾌감 포인트
- ✅ 1개 갈등 격화
- ✅ 1개 플롯 추진
- ✅ 대화 위주 (60% 이상)

**금지**:
- ❌ 3단락 초과 심리 묘사
- ❌ 5행 초과 환경 묘사
- ❌ 무의미한 일상 대화

---

### 엔딩 (후크)

**5종 결미 후크**:

#### 후크 1: 새 위기
```
이명이 일이 끝났다고 생각한 바로 그때.

멀리서 차가운 콧소리가 들려왔다.

"고작 금단이, 감히 방자하게?"

원영!
```

#### 후크 2: 신분 폭로
```
"너... 너는..."

그 사람이 눈을 크게 떴다.

"이 가문, 죽은 그 소주?"

이명이 냉소했다.

"이제, 너는 너무 많이 알았어."
```

#### 후크 3: 중요 정보
```
이명이 저물낭을 열었다.

안에 편지 한 장이 있었다.

내용을 확인한 순간, 안색이 변했다.

"그러니까, 사부님 말씀이 사실이었어..."
```

#### 후크 4: 강자 강림
```
"재미있군."

하늘에서 한 목소리가 떨어졌다.

이명이 고개를 들었다.

그 사람이 공중에 떠 있었다.

화신!
```

#### 후크 5: 예상치 못한 반전
```
이명이 막 안도의 한숨을 쉬었다.

갑자기, 가슴에 통증이 왔다.

고개를 숙여보니.

칼 한 자루가, 등 뒤에서 관통해 나와 있었다.

"미안해." 뒤에서 익숙한 목소리가 들렸다.

설마... 그녀?
```

## 쾌감 포인트 설계 공식

### 역전 공식 (7단계)

**표준 흐름**:

1. **의심**
```
"네가?" 장소가 냉소했다.
```

2. **조롱**
```
"연기기도 안 됐으면서, 감히 검을 드는 거야?"
주위에서 폭소가 터졌다.
```

3. **시현 (핵심)**
```
이명은 말이 없었다.
그저, 손을 들었다.

검이 울었다.
```

4. **초기 경악**
```
"이... 이 검명 소리는..."
누군가의 안색이 변했다.
```

5. **진실 공개**
```
"축기!"

"아니, 금단이야!"

"어떻게 가능해! 겨우 열여섯인데!"
```

6. **역전**
```
장소의 웃음이 얼굴에 굳었다.
```

7. **주인공 태연**
```
이명이 검을 거두고, 돌아섰다.

"비켜."

아무도 감히 막지 못했다.
```

---

### 과시 공식 (5단계)

1. **저조하게 숨김**
```
"새로 왔어?"
"응."
"수위가 어떻게 돼?"
"연기기."
```

2. **어쩔 수 없이 출수**
```
"네가 안 나서면, 내가 한다!"

상대방이 장을 날렸다.

이명이 한숨을 쉬었다.

"꼭 이래야 돼?"
```

3. **일격으로 제압**
```
손을 들었다.

가볍게 튕겼다.

상대방이 날아가 벽 3개를 뚫었다.
```

4. **모두 경악**
```
"이..."

"금단?"

"아니, 원영이야!"
```

5. **태연하게 떠남**
```
"미안, 손에 힘이 좀 들어갔어."

이명이 돌아서 갔다.

오직 바닥에 경악만 남겼다.
```

---

### 레벨업 공식 (6단계)

1. **강적을 만남**
```
"넌 내 상대가 안 돼."

상대방이 냉소했다.

맞다. 이명이 이를 악물었다. 격차가 너무 크다.
```

2. **엄청난 압박**
```
또 한 장.

이명이 날아가 피를 뱉었다.

정말, 여기서 죽는 건가?
```

3. **깨달음/기연**
```
아니.

갑자기 뭔가 떠올렸다.

사부님이 했던 말.

"검은, 원래 살육의 도구다."
```

4. **돌파**
```
쿵!

기운이 폭등!

금단 중기!

아직 안 끝났다!

금단 후기!

금단... 대원만!
```

5. **실력 시현**
```
이명이 일어섰다.

이번엔, 물러서지 않았다.

"이제, 내 차례야."
```

6. **역살**
```
일검.

오직 일검.

상대방이 쓰러졌다.
```

## 리듬 제어

### 타임라인 계획

**전기 (1-30장)**:
- 3장마다 1개 소클라이맥스
- 10장마다 1개 대클라이맥스
- 지속적 역전, 레벨업

**중기 (31-100장)**:
- 5장마다 1개 소클라이맥스
- 15장마다 1개 대클라이맥스
- 더 큰 음모 개입

**후기 (100장+)**:
- 10장마다 1개 소클라이맥스
- 20장마다 1개 대클라이맥스
- 최종 보스 등장

---

### 쾌감 밀도

**추천 구성**:
- 소쾌감 (역전, 과시): 챕터당 2-3개
- 중쾌감 (레벨업, 보물): 3-5장마다 1개
- 대쾌감 (반전, 역전): 10-15장마다 1개

## 금기

### ❌ 절대 금지

#### 1. 주인공 지능 저하
```
❌ 함정인 줄 알면서 뛰어듦
❌ 특정 스킬이 있다는 걸 잊음
❌ 무뇌 충동 경험치 제공
```

#### 2. 리듬 늘어짐
```
❌ 연속 3장 일상
❌ 10행 초과 수련 묘사
❌ 무의미한 심리 묘사
❌ 과도한 환경 묘사
```

#### 3. 조연 도구화
```
❌ 경험치 제공만을 위해 존재
❌ 배경판 역할만 함
❌ 합리적 동기 없음
❌ 지능이 갑자기 오프라인
```

#### 4. 패턴 남용
```
❌ 같은 역전 장면을 연속 2회 사용
❌ 영원히 "감히...내가 보기엔 죽고 싶은 모양이군"
❌ 무한 반복되는 "불가능"
```

## 대화 설계

### 빠른 리듬 대화

**특징**: 짧고, 빠르고, 강함

#### ✅ 예시
```
"너 누구야?"

"맞춰봐."

"죽고 싶어!"

"네가?"

"이—"

쾅!

끝.
```

---

### 과시 대화 템플릿

#### 템플릿 1: 반문식
```
"내가 누군지 아느냐!"

이명이 한번 봤다.

"몰라." 잠시 멈추고, "알고 싶지도 않고."
```

#### 템플릿 2: 태연식
```
"감히 내 동생을 죽여!"

"응." 이명이 고개를 끄덕였다, "그래서?"
```

#### 템플릿 3: 포스식
```
"널 죽이지 않을 이유 하나를 대봐."

"나... 내 아버지가..."

"부족해."

검이 집을 떠났다.
```

## 완전한 챕터 예시

### 장면: 주인공이 처음 두각을 나타냄

```
【오프닝 - 갈등】
"이명?"

장소가 냉소했다.

"파혼당한 폐물 말이지?"

【조롱 격화】
"연기기도 안 됐으면서 대비에 참가하겠다?"

주위에서 폭소가 터졌다.

이명은 거기 서서, 말이 없었다.

【어쩔 수 없이 출수】
"무릎 꿇어, 목숨을 살려줄 수도 있어." 장소가 다가왔다.

이명이 고개를 들었다.

"너는," 그의 목소리가 매우 차분했다, "자격 없어."

"죽고 싶어!"

장소가 일장을 날렸다.

금단 초기의 위압이 순간적으로 전장을 뒤덮었다.

【역전 - 시현】
하지만 다음 순간.

그의 손이 허공에 멈췄다.

정확히 말하면, 잡혔다.

"이게 다야?"

이명이 놓았다.

【역전 - 경악】
쿵!

장소의 온몸이 날아가 벽 3개를 뚫었다.

전장이 죽은 듯 고요했다.

"금단 중기..." 누군가 떨리며 입을 열었다.

"아니, 후기야!"

"어떻게 가능해! 겨우 스무살인데!"

【주인공 태연】
이명이 손의 먼지를 털었다.

"3년 전의 폐물은," 돌아서며, "지금의 너는, 건드릴 수 없어."

【결미 후크】
바로 그때.

멀리서 늙은 목소리가 들려왔다.

"재미있군. 금단 후기, 스무살..."

이명의 안색이 변했다.

이 목소리는...

원영!

게다가, 하나가 아니야!

(본 장 끝)
```

## 총결

**Fast-Paced 규범의 정수**:

1. **쾌감 밀집** - 챕터당 최소 2개
2. **빠른 리듬** - 질질 끌지 않기
3. **갈등 끊임없이** - 항상 긴장감
4. **강력한 후크** - 멈출 수 없게
5. **패턴은 새롭게** - 새로운 방식으로

---

**기억하세요**:

> 쾌감소설의 목적은 독자를 통쾌하게 하는 것,
> 한 챕터 한 챕터 멈출 수 없게 하는 것.
