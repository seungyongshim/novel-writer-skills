# 달달 규범

## 핵심 이념

**달달함 충분, 갈등 적절** - 독자가 달콤함에 만족하게 합니다.

---

## 핵심 요구

### 1. 달달함 밀도

**추천 구성**:
- 소달달 (소통, 관심): 챕터당 3-5개
- 중달달 (고백, 친밀): 3-5장마다 1개
- 대달달 (관계 확인, 중요 순간): 10장마다 1개

### 2. 달콤한 순간 설계

#### 유형 1: 일상적 소달달
```
"추워?"
아직 대답도 안 했는데, 외투가 이미 어깨 위에.
```

#### 유형 2: 따뜻한 디테일
```
그가 그녀가 고수를 안 먹는다는 걸 기억했다.
테이블 위의 음식에, 고수가 한 가닥도 없었다.
```

#### 유형 3: 사랑의 말
```
"있잖아?"
"응?"
"너를 만나고 나서야, 설렘이 뭔지 알았어."
```

### 3. 갈등 제어

**허용되는 갈등**:
- 작은 오해 (1-2장 내 해결)
- 외부 저항 (가족, 직장)
- 성격 조율

**금지되는 갈등**:
- 장기간 오해 (5장 초과)
- 제3자 개입 (진심으로 좋아함)
- BE 성향 전개

---

## 달달함 등급

- 미달달: 가끔 달달
- 중달달: 달달함 충분
- 과달달: 전 과정 하이 슈가
- 물릴 정도로 달달: 신중하게 사용
