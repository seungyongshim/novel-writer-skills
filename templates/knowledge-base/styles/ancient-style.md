# 고풍 집필 스타일

## 스타일 포지셔닝

**핵심 이념**: 고전의 운치와 현대의 감정, 의경을 우선으로

고전적 아름다움이 있되, 고전 문어체는 아니다; 시적이되, 난해하지 않다.

---

## 적용 시나리오

### ✅ 가장 적합
- 무협소설
- 선협소설 (문체 측면)
- 고대 배경 로맨스
- 고풍 판타지

### ⚠️ 신중하게 사용
- 현대 도시 (이질감이 큼)
- SF 소재

---

## 핵심 원칙

### 1. 구어체 위주, 고전적 운치 곁들임

**고전 문어체가 아니라, 고전적 운치가 담긴 구어체**

#### ❌ 순수 문어체 (너무 어려움)
```
완청이 검을 잡고 서 있었다, 바람이 불어 옷자락이 펄럭이고, 눈빛이 날카로워, 정면을 직시하였다.
적이 이르자, 검광이 갑자기 나타나, 용이 못에서 솟구치듯, 그 기세를 막을 수 없었다.
```

#### ✅ 고전적 운치의 구어체
```
완청이 거기 서 있었다, 손에 검을 쥐고.
바람이 그녀의 옷자락을 불어올렸다.
그녀는 앞의 사람을 보며, 말하지 않았다.

검이, 칼집에서 나왔다.

한 줄기 빛이 스쳤다, 번개처럼 빠르게.
```

**요점**:
- 어휘는 고전적 (옷자락, 출초)
- 문형은 현대적 (그녀, 말하지 않았다)
- 결합 사용

---

### 2. 풍부한 의상

**의상으로 직접적 묘사를 대체**

#### ❌ 직접적 묘사
```
그녀는 매우 슬퍼, 혼자 오래 울었다.
```

#### ✅ 의상적 묘사
```
그녀가 창가에 홀로 앉았다.
촛불이 흔들리고, 그림자가 벽에서 흔들렸다.
눈물이, 거문고 줄 위에 떨어졌다.

거문고, 소리가 없다.
```

**자주 쓰는 의상**:
- 바람, 비, 눈, 달
- 검, 거문고, 바둑, 책
- 꽃, 잎, 구름, 연기

---

### 3. 여백과 시적 정취

**고풍의 정수는 여백**

#### ✅ 예시
```
그가 떠났다.

바람은 여전히 불고 있다.
잎은 여전히 떨어지고 있다.

그녀는 아직 거기 서 있다.

기다린다.

아니면 기다리지 않는 걸까?

(여백, 명시하지 않음)
```

## 어휘 선택

### 고전적 어휘

**현대 어휘를 고전 어휘로 대체**

| 현대 어휘 | 고전 어휘 |
|--------|--------|
| 옷 | 옷자락, 장삼, 라군 |
| 머리카락 | 청사, 쪽머리 |
| 눈(안구) | 눈동자, 안목 |
| 손 | 옥수, 섬수 (여성) |
| 걷다 | 거닐다, 천천히 걷다, 느린 걸음 |
| 말하다 | 이르다, 읊다, 입을 열다 |
| 웃다 | 미소짓다, 얕은 웃음, 실소 |
| 보다 | 바라보다, 살피다, 곁눈질하다 |

---

### 현대적 어휘 금지

#### ❌ 절대 금지
```
- OK, 바이바이, 하하
- 컴퓨터, 핸드폰, 자동차
- 매우, 엄청, 특별히 (과도한 사용)
- 멋지다, 쿨하다, 귀엽다
```

#### ✅ 고풍 대체
```
- 좋소, 그렇소, 응
- 강호, 무림, 중원
- 심히, 꽤, 극히
- 준수하다, 청랑하다, 사랑스럽다
```

## 문형 특징

### 시적 단문

**고풍 단문에는 운치가 있어야**

#### ✅ 예시 1
```
완청이 검을 잡고 서 있다.
바람이, 불었다.
그녀의 옷자락이, 펄럭인다.

검광이, 서리처럼.
```

#### ✅ 예시 2
```
그가 웃었다.
희미하게 웃었다.

"네가, 그래도 왔구나."

"응."

그녀가 고개를 끄덕이며, 더 말하지 않았다.
```

---

### 대구와 배비

**적절히 대구를 사용해, 운율감을 더한다**

#### ✅ 예시
```
검이, 손에.
마음이, 검에.

사람은 아직 이르지 않았으나, 검이 먼저 도착했다.
초식은 아직 나오지 않았으나, 기세가 이미 드러났다.
```

**주의**:
- 과도하게 사용하지 않을 것
- 2-3구절이면 충분
- 중요한 순간에 사용

## 대화 설계

### 고풍 대화

**고전적 운치를 유지하되, 어색하지 않게**

#### ❌ 지나친 문어체
```
"각하는 누구시며, 어찌 여기서 내 길을 막으시오?"
"소인은..."
```

#### ✅ 고전적 운치의 구어체
```
"넌 누구지?"

"맞춰봐." 그가 웃었다.

"맞출 겨를이 없어." 그녀가 검을 들며, "비켜."

"비키지 않으면?"

"그렇다면, 칼 아래서 진위를 가리지."
```

**요점**:
- "너", "나"를 써도 됨 ("각하", "소인"을 꼭 쓸 필요 없음)
- 고풍 습관 용어 유지
- 대화는 간결하고 힘 있게

---

### 호칭 체계

**관계에 따라 다른 호칭 사용**

#### 동배 사이
```
- 남성: 형대, 형제, 인형
- 여성: 낭자, 여협
- 친밀: 너, 이름 직접 부름
```

#### 어른
```
- 사부, 선배, 노선배
- 장문인, 종주
```

#### 후배
```
- 꼬마, 아가씨, 아이
```

## 장면 묘사

### 강호 장면

#### ✅ 예시: 객잔
```
객잔에 사람이 적지 않았다.

구석에 복면인이 앉아 있었다, 술 한 병, 혼자.
창가에 누군가 바둑을 두고 있었다, 돌 놓는 소리가 고요했다.
카운터 뒤에서, 주인이 주판을 튕기고 있었다, 딱딱딱 소리가 났다.

완청이 문을 밀고 들어왔다.

객잔의 사람들이, 모두 그녀를 한번 쳐다보았다.
그리고 다시 각자의 일을 했다.

강호의 규칙.
```

---

#### ✅ 예시: 대나무 숲
```
대나무 숲 깊숙이, 작은 집이 한 채 있었다.

청석이 깔리고, 대나무 그림자가 흔들리고.
뜰에 한 사람이, 검을 연습하고 있었다.

검광이 번쩍이며, 바람 소리를 일으켰다.
대나무 잎이 분분히 떨어지며, 검광 속에서 날았다.

검을, 거두다.

대나무 잎이, 땅에 떨어지다.

모든 것이 고요해졌다.

오직 바람만, 아직 불고 있었다.
```

---

### 무투 장면

**고풍 무투에는 화면감이 있어야**

#### ✅ 예시
```
검이, 칼집에서 나왔다.

한 줄기 빛이 스쳤다.

상대가 몸을 비틀어, 간신히 피했다.
검봉이 그의 소매를 스치며, 한 줄기 갈라짐을 남겼다.

"빠른 검이로구나!"

그가 말을 채 끝내지 못했다.

두 번째 검이, 이미 도착했다.

이번에는 피하지 못했다.

검이, 그의 목 앞 세 촌에서 멈췄다.

"졌다."

그가 쓴웃음을 지었다, "그래, 졌어."

검이, 칼집으로 돌아갔다.
```

**요점**:
- 단문으로 리듬 조성
- 단독 단락으로 충격 강화
- 여백 (검이 어떻게 나왔는지? 독자가 상상)

## 감정 묘사

### 함축적 표현

**고풍 감정은 함축적이어야**

#### ❌ 직접적 표현
```
"사랑해."
"나도 사랑해."
```

#### ✅ 함축적 표현
```
"만약 다음 생이 있다면, ...하시겠어요?"

그녀가 말을 끝내지 못했다.

그도 그녀가 끝내지 못하게 했다.

"응."

한 글자, 이미 약속이었다.
```

---

### 의상으로 감정 전달

**의상으로 감정을 표현한다**

#### ✅ 예시: 달
```
그가 달을 바라보았다.

둥글다.

그런데 왜, 마음에는 한 조각이 빠진 것 같을까?

그녀도 달을 보고 있을까?

아닐 거다.

어차피, 그녀는 벌써 떠났으니까.

달은 여전히 그 달이다.
사람은 더 이상 그 사람이 아니다.
```

## 시대감 조성

### 생활 디테일

**디테일로 시대감을 조성한다**

#### ✅ 예시
```
그녀가 동경 앞에 앉아, 머리를 빗고 있었다.
청사가 폭포처럼, 한 번에 끝까지 빗겼다.

시녀가 더운 물을 가져왔다.

"아가씨, 옷을 갈아입으셔야 합니다."

"응."

그녀가 일어서자, 시녀가 허리띠를 풀어주었다.
라군이 바닥에 떨어지고, 날렵한 복장으로 갈아입었다.

"오늘은 치마를 안 입으세요?"

"응," 그녀가 허리띠를 매며, "외출해야 해서."
```

**요점**:
- 동경, 시녀, 라군, 경장
- 모두 고대 생활 디테일
- 자연스럽게 서사에 녹아듦

---

### 예절 습관

**고대 예절에 맞춰야**

#### ✅ 예시
```
그가 그녀에게 포권을 했다.

"예를 갖추겠습니다."

그녀가 가볍게 고개를 숙여, 답례로 삼았다.

(주: 남자는 포권, 여자는 고개 숙임/무릎을 굽힘)
```

## 흔한 장면

### 장면 1: 첫 만남

#### ✅ 고풍 버전
```
살구꽃비 속에서 그를 만났다.

그가 유지 우산을 받치고, 다리 위에 서 있었다.
백의가 눈처럼 하얗고, 마르고 곧았다.

바람이 불어, 그의 옷자락을 일으켰다.

그녀가 발걸음을 멈추었다.

그도 고개를 들어, 그녀를 바라보았다.

눈이 마주쳤다.

아무도 먼저 입을 열지 않았다.

살구꽃이 떨어져, 그녀의 어깨에 내려앉을 때까지.

그가 웃었다.

"낭자, 우산 아래 한 사람 더 들어갈 수 있겠습니까?"

그녀는 그를 보며, 말하지 않았다.

하지만 우산 반쪽의 자리를 비워주었다.

그가 다가와, 그녀와 나란히 섰다.

비는 여전히 내리고 있었다.

그들은 그렇게, 함께 그 다리를 건넜다.
```

**요점**:
- 의상 (살구꽃비, 유지 우산, 백의)
- 함축 (좋아한다고 직접 말하지 않음, 하지만 자리를 비워줌)
- 시적 정취 (빗속에 나란히 걷기)

---

### 장면 2: 이별

#### ✅ 고풍 버전
```
"정말 가시는 거예요?"

"응."

"언제 돌아오시나요?"

"모릅니다."

"그럼..." 그녀가 잠시 머뭇거리며, "가시는 길에 평안하세요."

"응."

그가 돌아서, 몇 걸음 걷다가, 다시 돌아보았다.

"기다려줘요."

두 글자.

그녀가 고개를 끄덕였다.

"좋아요."

역시 두 글자.

이번에는 정말 떠났다.

그녀는 제자리에 서서, 그의 뒷모습이 길 끝에서 사라지는 것을 바라보았다.

바람이 불었다.

만땅의 낙엽을 일으키며.

그녀가 돌아서, 집으로 들어갔다.

거문고, 아직 있다.
검, 아직 있다.

사람, 없다.
```

## AI 느낌 제거 규칙

### 고풍에서도 AI 느낌 제거

#### ❌ 금지 어휘
```
- 그러나, 뜻밖에도, 예상치 못하게
- 마치...처럼
- 오미잡진, 백감교집
- 극도로, 만분, 이상하게
```

#### ✅ 고풍 표현
```
- 하지만, 그런데, 다만
- ~같은, ~처럼, 마치
- 마음속에 만 가지 생각이
- 심히, 꽤, 극히
```

---

### 나열 금지

#### ❌ 나열
```
그녀의 피부는 응지 같고, 눈썹은 먼 산 같고, 눈동자는 가을 물 같고, 입술은 홍점 같고,
날씬한 몸매가 버들 같고, 걸음걸이가 가벼워 제비 같았다.
```

#### ✅ 절제
```
그녀는 예뻤다.

얼마나 예뻤냐면?

한 번만 보면,
이 생에 잊을 수 없을 정도로.
```

## 총결

**Ancient-Style 스타일의 정수**:

1. **고전의 운치와 현대의 감정** - 고전적 아름다움이 있되, 난해하지 않게
2. **의상을 우선** - 의상으로 직접적 서술을 대체
3. **함축적 표현** - 감정은 절제해야
4. **여백의 예술** - 다 말하지 않음
5. **시적 리듬** - 언어에 운율이 있어야

---

**기억하세요**:

> 고풍은 화려한 수사를 나열하는 것이 아니라,
> 가장 단순한 문자로, 가장 깊고 먼 의경을 만드는 것입니다.
