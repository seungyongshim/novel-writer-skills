# 미니멀 집필 스타일

## 스타일 포지셔닝

**핵심 이념**: 모든 불필요한 것을 삭제하고, 필수만 남긴다

헤밍웨이의 "빙산 이론"에서 영감 - 수면 위는 문자, 수면 아래는 여백.

---

## 적용 시나리오

### ✅ 가장 적합
- 미스터리/추리
- 하드보일드 소설
- 실험 문학
- 미니멀리즘 작품

### ⚠️ 신중하게 사용
- 섬세한 감정 묘사가 필요한 로맨스
- 웅장한 장면이 필요한 서사시

---

## 핵심 원칙

### 1. 모든 수식어 삭제

**극단적 원칙**: 삭제할 수 있으면 삭제

#### ❌ 수식어 있음
```
그녀는 천천히 낡은 나무 문을 밀었다.
```

#### ✅ 미니멀 버전
```
그녀가 문을 밀었다.
```

---

### 2. 짧은 문장이 왕

**80%의 문장이 10자를 넘지 않는다**

#### ✅ 예시
```
임완청이 문을 밀었다. 어둡다.

"왜 불을 안 켰어?"
"잊었어."
"아파?"

그녀가 이마를 짚었다. 뜨겁다.
```

**특징**:
- 문장마다 3-8자
- 단독으로 단락 구성
- 리듬이 단단함

---

### 3. 무설명

**동기를 설명하지 않고, 감정을 설명하지 않는다**

#### ❌ 설명 버전
```
그녀는 화가 났다, 그가 또 생일을 잊었으니까.
```

#### ✅ 미니멀 버전
```
그녀가 전화를 끊었다.
```

(여백: 왜 끊었을까? 독자가 판단)

---

## 문형 규범

### 문형 제한

**3가지 문형만 허용**:

1. **서술문** (주어+서술어+목적어)
```
그녀가 앉았다.
그가 떠났다.
비가 멈췄다.
```

2. **대화**
```
"갈까?"
"가자."
```

3. **단어/한 글자**
```
어둠.
비.
추위.
```

---

### 금지 문형

#### ❌ 완전 금지
- 비유문 (~처럼, ~같은, 마치)
- 대구문
- 설의문
- 감탄문 (과도한 사용)
- 종속절 (~때문에...그래서...)

---

## 대화 설계

### 미니멀 대화

**특징**: 짧고, 직접적이며, 설명 없음

#### ✅ 예시 1
```
"밥 먹었어?"
"응."
"뭐 먹었어?"
"면."
```

#### ✅ 예시 2
```
"갈까?"
"가자."

그녀가 일어섰다. 그도 일어섰다.

문. 밖. 비.

"잠깐?"
"안 기다려."
```

---

### 대화 태그

**원칙**: 생략할 수 있으면 생략

#### ❌ 태그 있음
```
"왜 그래?" 그녀가 물었다.
"아무것도 아니야." 그가 말했다.
```

#### ✅ 태그 없음
```
"왜 그래?"
"아무것도 아니야."
```

---

## 묘사 기법

### 환경 묘사

**극도의 절제, 필요한 것만 쓴다**

#### ❌ 전통적 묘사
```
카페에 사람이 많지 않고, 조명이 어두웠으며,
구석에서 재즈 음악 소리가 들려왔다.
```

#### ✅ 미니멀 버전
```
카페. 사람 적음. 어두움.
```

---

### 감정 묘사

**행동만 쓰고, 감정은 쓰지 않는다**

#### ❌ 감정 직접 서술
```
그녀는 매우 슬퍼서, 눈물을 멈출 수 없었다.
```

#### ✅ 행동만 서술
```
그녀가 고개를 숙였다.
어깨가 떨리고 있다.
```

(여백: 슬픈 건가? 화난 건가? 무서운 건가? 독자가 느낀다)

---

### 동작 묘사

**동사 하나면 충분하다**

#### ❌ 상세한 묘사
```
그녀는 조심스럽게 문을 밀어 열고, 고개를 내밀어 안을 들여다보았다.
```

#### ✅ 미니멀 버전
```
그녀가 문을 밀었다. 고개를 내밀었다.
```

---

## 단락 구조

### 단문 단락

**Minimal 스타일의 상징**

#### ✅ 예시
```
그녀가 깨어났다.

날이 밝았다.

그가 없다.

침대가 차갑다.

그녀가 일어났다. 옷을 입었다.

거울 속 자신. 낯설다.

그녀가 문을 나섰다.
```

**요점**:
- 한 문장이 하나의 단락
- 멈춤의 감각을 만듦
- 리듬을 강화

---

### 연속 단문

**연속 단문으로 긴장감을 만든다**

#### ✅ 예시
```
문.
열렸다.
그였다.
그녀가 멈칫했다.
"너..."
"알아."
그녀가 뒷걸음질쳤다.
그가 다가왔다.
그녀가 벽에 닿았다.
물러설 곳이 없다.
```

## 여백의 예술

### 말하지 않기

**가장 중요한 것은, 쓰지 않는다**

#### ✅ 예시 1: 감정의 여백
```
"아직 나를 사랑해?"

그는 말하지 않았다.

그녀도 더 이상 묻지 않았다.
```

(여백: 답이 뭘까? 독자가 판단)

---

#### ✅ 예시 2: 플롯의 여백
```
그녀가 방에 들어갔다.

10분 후.

그녀가 나왔다.

얼굴에 붉은 자국이 있었다.
```

(여백: 무슨 일이 있었을까? 독자가 상상)

---

#### ✅ 예시 3: 결말의 여백
```
그가 총을 들었다.

탕.

그녀가 쓰러졌다.

아니면 그가 쓰러졌을까?

(장 끝)
```

## 문장부호 사용

### 미니멀 문장부호

**가장 기본적인 문장부호만 사용**

**사용 허용**:
- 마침표 .
- 물음표 ?
- 쉼표 , (필요할 때)
- 따옴표 "" (대화)

**사용 금지**:
- 줄임표 ......
- 대시 ——
- 느낌표!!! (쓸 수 있지만 절제)
- 세미콜론 ;

---

### 빈 줄의 힘

**빈 줄로 문장부호를 대체한다**

#### ✅ 예시
```
그녀가 떠났다

그는 그녀의 뒷모습을 바라보았다

거리 모퉁이에서 사라졌다

그가 담배를 꺼냈다

피우지 않았다

껐다
```

## 완전한 장면 예시

### 장면: 이별

#### ✅ Minimal 스타일 버전
```
"우리 헤어지자."

그녀가 그를 보았다.

그는 말하지 않았다.

"들었어?"

"들었어."

"그럼..."

"그래."

그가 일어섰다.

외투. 열쇠.

"이게 끝이야?"

"응."

문.

열림.

닫힘.

그녀 혼자.

울었다.

아니면 울지 않았을까?

모른다.

어쨌든.

날이 어두워졌다.
```

**분석**:
- 98%가 단문 (10자 이내)
- 형용사 제로
- 거대한 여백 (그의 표정은? 그녀가 진짜 울었을까?)
- 독자가 스스로 세부사항을 채움

---

## 금기사항

### ❌ 절대 있어서는 안 되는 것

1. **모든 형용사**
   - 아름다운, 슬픈, 분노한 등
   - 전부 삭제

2. **모든 비유**
   - ~처럼, 마치, ~같은
   - 하나도 쓰지 않는다

3. **심리 묘사**
   - 그녀는 생각했다, 그는 느꼈다
   - 행동만 쓴다

4. **환경 분위기 조성**
   - 불필요한 환경 묘사
   - 필요한 것만 쓴다
