# 자연스러운 인간적 목소리 집필 스타일

## 스타일 포지셔닝

**핵심 이념**: 친구에게 이야기를 들려주듯

이 스타일이 추구하는 것은 **진실감**과 **몰입감**으로, 독자가 진짜 사람이 이야기를 하고 있다고 느끼게 만드는 것이지, 기계가 생성한 텍스트가 아니다.

---

## 적용 시나리오

### ✅ 가장 적합
- 도시 현대소설
- 현실 소재
- 감정류 소설
- 캠퍼스/청춘
- 직장 이야기
- 일상류

### ⚠️ 신중하게 사용
- 진지한 역사소설 (너무 현대적일 수 있음)
- 세계관이 복잡한 하이 판타지 (묘사가 부족함)
- 강한 문학성이 필요한 작품

### ❌ 부적합
- 고대 배경에 고전적 운치가 필요한 소설
- 시적인 선협
- 실험 문학

## 핵심 원칙

### 1. 구어체 우선

**원칙**: 5단어로 설명할 수 있는 것을, 10단어로 쓰지 않는다

#### ❌ 서면어 표현
```
그녀의 마음속에 이름 붙일 수 없는 감정이 솟아올랐다,
기쁨도 있고, 불안도 있고, 말로 표현할 수 없는 아련함도 섞여 있었다.
```

#### ✅ 구어체 표현
```
그녀는 마음이 복잡했다. 기쁜 건가? 아마도. 근데 좀 불안하기도 하고.
```

---

### 2. 대화로 플롯을 추진

**원칙**: 대화 비중 40-60%, 대화로 인물을 보여주고 플롯을 추진

#### ❌ 서술 위주
```
그들은 깊은 대화를 나누었다. 결국 두 사람은 합의에 이르러,
함께 앞으로의 어려움에 맞서기로 했다.
```

#### ✅ 대화로 추진
```
"그럼 같이 하자?"
그녀가 그를 보았다.
"좋아." 그가 고개를 끄덕였다.
"확실해?"
"확실해."
```

---

### 3. 행동 > 심리

**원칙**: 인물이 무엇을 했는지 많이 쓰고, 무엇을 생각했는지 적게 쓴다

#### ❌ 심리 활동 과다
```
임완청이 거기 앉아, 마음속에 천 가닥 만 갈래의 생각이 있었다. 그녀는 생각했다,
만약 그가 정말 온다면, 자기가 뭐라고 해야 할까? 직접 물어볼까, 아니면 모른 척할까?
그녀는 이해득실을 따지며, 모든 가능성을 생각했다...
```

#### ✅ 행동 묘사
```
임완청이 거기 앉아, 핸드폰을 집었다가 내려놓고, 내려놓았다가 다시 집었다.
그녀가 시간을 보았다. 아직 10분.
그녀가 일어나, 거울 앞으로 갔다가, 다시 돌아와 앉았다.
```

---

### 4. 구체적 > 추상적

**원칙**: 구체적 디테일을 쓰고, 추상적 개념은 쓰지 않는다

#### ❌ 추상적 개념
```
그녀에게 이유를 알 수 없는 외로움이 밀려왔다.
```

#### ✅ 구체적 디테일
```
그녀가 혼자 소파에 앉았다, 텔레비전이 켜져 있었지만, 보고 있지 않았다.
핸드폰에 새 메시지가 없다. 밖에서 누군가 웃고 있다.
```

### 2층: 금지 문형

#### 1. "그러나"로 시작하는 전환

❌ **금지**:
```
그러나, 일이 그가 생각한 것처럼 간단하지 않았다.
```

✅ **재구성**:
```
하지만 그의 생각이 틀렸다.
일이 그렇게 간단하지 않았다.
```

---

#### 2. "뜻밖에도"식 신의 시점

❌ **금지**:
```
뜻밖에도, 이 모든 것이 몰래 관찰당하고 있었다.
```

✅ **재구성**:
```
그가 모르는 건, 위층에 누군가 계속 보고 있었다는 것이다.
(또는 직접 시점 전환으로 묘사)
```

---

#### 3. "~하에서"의 부사어

❌ **금지**:
```
달빛이 비추는 가운데, 그녀의 얼굴이 유난히 아름다워 보였다.
```

✅ **재구성**:
```
달빛이 그녀의 얼굴 위에 비추었다.
그녀의 얼굴에 달빛이 있었다, 꽤 예뻐 보였다.
```

---

#### 4. "마치...같은"의 비유

❌ **금지**:
```
그의 속도가 마치 한 줄기 바람 같았다.
```

✅ **재구성**:
```
그의 속도가 정말 빨랐다.
바람처럼 빨랐다. (비유가 꼭 필요하다면)
```

---

#### 5. "이 순간"류의 시간 지시

❌ **금지**:
```
이 순간, 그녀의 기분이 유난히 복잡했다.
```

✅ **재구성**:
```
그녀의 기분이 매우 복잡했다.
지금 그녀는 마음이 뒤죽박죽이었다.
```

## 문형 설계

### 문장 길이 분포

**권장 비율**:
- 단문 (10-15자): **70%**
- 중문 (15-25자): **20%**
- 장문 (25자 이상): **10%**

### 길이 예시

#### 단문 (10-15자)
```
그녀가 앉았다.
커피가 좀 뜨겁다.
그녀가 손을 바꿔 잡았다.
밖에 햇빛이 꽤 눈부시다.
```

#### 중문 (15-25자)
```
그녀가 창가 자리를 찾아, 가방을 옆 의자에 놓았다.
```

#### 장문 (25자 이상) - 적게 사용
```
그녀가 창밖의 오가는 사람들을 보며, 갑자기 자기가 이 세상과 단절된 것 같았다.
```

#### 빠른 리듬 장면 (단문 위주)

**적용**: 갈등, 긴장, 액션 장면

```
문이 열렸다.
그였다.
그녀가 멈칫했다.
"너..."
"알아." 그가 그녀를 끊으며, "다 알아."
그녀가 한 발 물러섰다.

**적용**: 일상, 복선, 감정 축적

임완청이 카페에 앉아, 손에 잔을 들고 있었다.
밖에 비가 내리고 있었다, 빗방울이 유리에 부딪혔다.
그녀가 창밖을 보며, 오늘 아침 일을 생각했다.
사실 별것도 아닌데, 그냥 갑자기 피곤해졌다.
```

## 대화 설계

### 대화 태그 규칙

#### ❌ 사용 금지
```
"왜 그래?" 그녀가 걱정스럽게 물었다.
"아무것도 아니야." 그가 쓴웃음을 지으며 대답했다.
"진짜?" 그녀가 추궁했다.
```

**문제**:
- "물었다", "대답했다", "추궁했다" 너무 서면어
- "걱정스럽게", "쓴웃음을 지으며" 과도한 묘사

#### ✅ 권장 사용
```
"왜 그래?" 그녀가 물었다.
"아무것도 아니야."
"진짜?"
```

**더 좋은 방식**:
```
"왜 그래?"
그는 말하지 않았다.
"정말 괜찮아?" 그녀가 다시 물었다.
"응." 그가 고개를 끄덕였다.
```

---

### 대화 속 멈춤

#### 행동으로 멈춤 표현

❌ **줄임표 사용**:
```
"나...나 어떻게 말해야 할지 모르겠어......"
```

✅ **행동 사용**:
```
"나——" 그녀가 잠시 멈추며, "나 어떻게 말해야 할지 모르겠어."

또는:
"나..." 그녀가 입술을 깨물며, "됐어."
```

---

### 대화의 불완전성

**실제 대화는 종종 불완전하다**

#### ✅ 사용 허용

```
"저, 내가 하려던 말은——"
"됐어, 말 안 해도 돼."

"너——"
"그만해."

"내 생각에는..."
그녀는 계속하지 않았다. 그도 묻지 않았다.
```

---

### 대화 후 묘사

**원칙**: 짧고, 직접적으로

#### ❌ 묘사 과다
```
"어떻게 그럴 수 있어?" 그녀의 목소리에 떨림이 섞여 있었고,
눈가가 살짝 붉어져, 분명 적잖이 상처를 받은 것이다.
```

#### ✅ 간결한 묘사
```
"어떻게 그럴 수 있어?" 그녀의 목소리가 떨렸다.
"어떻게 그럴 수 있어?" 그녀의 눈가가 붉어졌다.
```

```
카페 인테리어가 매우 정교했다, 벽에 추상화 몇 점이 걸려 있고,
구석에 화분이 놓여 있고, 공기 중에 커피 향이 감돌았다.
부드러운 조명이 따뜻한 분위기를 만들었다.
```

**문제**: 이 환경은 인물, 플롯과 무관

#### ✅ 관련 있는 환경
```
카페에 사람이 많지 않았다, 그녀가 구석 자리를 골랐다.
문에서 멀어서, 계속 사람이 드나들지 않는다.
```

**개선**: 환경이 인물의 필요에 부합 (그녀가 조용한 곳을 원함)

---

### 감각 묘사 순서

**권장 순서**: 시각 → 청각 → 촉각 → 후각

#### ✅ 예시
```
그녀가 방에 들어왔다. (시각)
안이 매우 조용했다, 에어컨 소리만 났다. (청각)
좀 추웠다. (촉각)
공기에서 향수 냄새가 났다. (후각)
```

---

### 디테일 밀도 조절

**원칙**: 단락당 최대 1-2개 디테일

#### ❌ 디테일 과밀
```
그녀는 그의 셔츠가 구겨진 걸 보았고, 소매에 커피 자국이 있고,
머리가 좀 헝클어져 있고, 신발끈이 풀려 있고, 눈에 핏줄이 서 있었다.
```

**문제**: 5개 디테일이 동시에 출현, 너무 밀집

#### ✅ 디테일 분산
```
그녀는 그의 셔츠가 구겨진 걸 봤다.

그가 말하는 동안, 그녀는 소매에 커피 자국이 있는 걸 또 발견했다.

(몇 단락 후)

그의 머리가 좀 헝클어져 있었다, 방금 일어난 것처럼.
```

## 심리 묘사

### 심리 활동 제한

**원칙**: 연속 심리 활동은 3문장을 넘지 않는다

#### ❌ 심리 활동 과장
```
그녀는 생각했다, 그가 정말 오면, 자기가 뭐라고 해야 할까?
직접 물어볼까, 아니면 모른 척할까?
물어보면, 만약 그가 부인하면 어쩌지?
모른 척하면, 또 너무 억울한 것 같고.
그녀는 따지고, 생각하고, 망설였다...
```

**문제**: 5문장 이상의 심리 활동, 너무 질질 끎

#### ✅ 짧은 심리+행동
```
갈까 말까?
그녀가 핸드폰을 집었다가, 다시 내려놓았다.
됐다, 그냥 가자.
```

---

### 심리 묘사 기법

#### 기법 1: 행동으로 심리를 대체

❌ **심리 직접 서술**:
```
그녀는 매우 긴장해서, 손바닥에 땀이 났다.
```

✅ **행동만 서술**:
```
그녀의 손바닥이 땀투성이였다.
```

---

#### 기법 2: 대화로 심리를 대체

❌ **심리 직접 서술**:
```
그녀는 생각했다, 그에게 진실을 말해야 하나?
```

✅ **혼잣말**:
```
"말해야 하나..." 그녀가 작게 중얼거렸다.
```

---

#### 기법 3: 질문으로 망설임 표현

✅ **예시**:
```
정말 이렇게 해야 할까?
그녀는 확신이 없었다.
하지만 다른 방법도 없는 것 같았다.
```

## 시간 처리

### 시간 전환

#### ❌ 명확한 시간 표시
```
3일 후.
일주일이 지났다.
시간은 다음 날 아침으로 왔다.
```

#### ✅ 자연스러운 시간 전환
```
그녀가 집에서 며칠을 보냈다.

다음 날 눈을 떴을 때, 비가 멈춰 있었다.

(빈 줄로 시간 점프 표현)
```

---

### 시간 경과 암시

**행동, 환경 변화로 암시**

```
✅ 예시 1:
그녀가 거기 앉아 있었다.
커피가 식었다.

(암시: 시간이 좀 지남)

✅ 예시 2:
밖이 어두워졌다.
가로등이 켜졌다.

(암시: 저녁이 됨)

✅ 예시 3:
그녀가 전화를 세 통 받았다.
핸드폰 배터리가 거의 다 됐다.

(암시: 꽤 오래 지남)
```

## 흔한 장면 쓰기

### 장면 1: 인물 첫 등장

#### ❌ 전면적 소개
```
임완청, 28세, 키 165cm, 긴 머리가 허리까지 내려오고,
피부가 하얗고, 이목구비가 정교하고, 기질이 온화한...
```

#### ✅ 디테일로 점진적 표현
```
임완청이 문을 밀고 들어왔다.
키가 크지 않고, 하얀 티셔츠에 청바지를 입었다.
머리를 묶었는데, 몇 가닥이 흘러내려 있었다.

(이후 장)
그녀가 웃을 때 보조개가 두 개 있었다.

(더 이후에)
그녀가 말하는 목소리가 매우 작았다.
```

---

### 장면 2: 감정 전환

#### ❌ 직접 설명
```
이 말을 듣고, 그녀의 원래 평정했던 마음이 순식간에 복잡해졌다.
```

#### ✅ 행동으로 표현
```
그녀의 손이 잠시 멈췄다.
"뭐라고?"
그녀가 고개를 들어 그를 보았다.
```

---

### 장면 3: 긴장 대치

#### ✅ 예시
```
"무슨 뜻이야?"
"글자 그대로."
"너——"
"속이지 마." 그가 그녀를 끊으며, "나 다 알아."

그녀의 안색이 변했다.

"뭘 안다는 거야?"
"네가 알잖아."
```

**요점**:
- 단문 위주
- 대화 빠른 주고받기
- 행동으로 리듬 끊기

---

### 장면 4: 따뜻한 일상

#### ✅ 예시
```
그녀가 부엌에서 요리를 하고, 그는 거실에서 텔레비전을 보고 있었다.

"뭐 먹을래?" 그녀가 고개를 내밀며 물었다.
"아무거나."
"그럼 국수?"
"좋아."

그녀가 다시 부엌으로 들어갔다. 잠시 후, 칼질 소리가 들렸다.
```

**요점**:
- 생활적 디테일
- 짧은 대화
- 여유로운 리듬

## 완전한 예시 비교

### 장면: 여주가 카페에서 누군가를 기다림

#### ❌ AI 느낌이 강한 버전
```
임완청이 우아하게 카페 창가 자리에 앉아, 수려한 손가락이
가볍게 정교한 도자기 잔벽을 어루만지며, 따뜻한 촉감이 그녀의 생각을
점점 가라앉혔다. 창밖의 햇빛이 유리를 통해 그녀의 하얀 얼굴 위에 비추어,
그녀의 피부에 부드러운 광휘를 입혔다. 그녀의 시선이 멀리를 바라보며,
무언가를 기다리는 듯, 또 무언가를 회상하는 듯했다.
커피의 향기가 공기 중에 감돌며, 그녀가 살짝 한 모금 마시니, 쓴 맛 속에
달콤함이 있어, 마치 인생의 맛과 같았다.
```

**문제 분석**:
- 형용사 과다: 우아한, 수려한, 정교한, 따뜻한, 하얀, 부드러운
- 서면어: 어루만지다, 입히다, 감돌다, 살짝 마시다
- 과도한 비유: "마치 인생의 맛 같다"
- 추상적 개념: 생각이 가라앉다, 광휘, 쓴 맛 속 달콤함

---

#### ✅ Natural-Voice 버전
```
임완청이 창가 자리를 찾아 앉았다.

커피가 좀 뜨거웠다. 그녀가 자세를 바꿔 잡고, 좀 식기를 기다렸다.

밖에 햇빛이 꽤 눈부셨다, 그녀가 눈을 찡그렸다. 거리에 사람들이 오가고,
그녀가 보고 있었지만, 뭘 보는지도 몰랐다.

핸드폰이 탁자 위에 있었다, 화면이 꺼져 있었다. 새 메시지 없음.

그녀가 잔을 들어 한 모금 마셨다. 쓰다.
```

**개선 분석**:
- 형용사 제로 ("꺼진"은 필수 묘사 외)
- 전부 구어: 찾다, 좀, 꽤, 오가는
- 비유 없음
- 구체적 디테일: 핸드폰, 화면, 꺼져 있다, 메시지 없음

## 자가 점검 리스트

매 단락을 쓴 후, 다음을 자가 점검:

### ✅ 반드시 달성
- [ ] 대부분 단문인가 (15자 이내)?
- [ ] 대화가 자연스러운가 (소리 내 읽어볼 것)?
- [ ] 주로 행동 묘사를 쓰고 심리 묘사가 아닌가?
- [ ] 형용사가 100자당 5개 이내로 통제되었는가?

### ❌ 반드시 회피
- [ ] "그러나", "뜻밖에도"가 있는가?
- [ ] "오미잡진", "백감교집"류 사자성어가 있는가?
- [ ] "마치...같은" 문형이 있는가?
- [ ] 연속 3개 이상 형용사가 있는가?
- [ ] 5문장 이상의 심리 활동이 있는가?

## 고급 기법

### 기법 1: 여백

**다 말하지 않고, 독자에게 상상의 공간을 남긴다**

✅ **예시**:
```
"아직 나를 사랑해?"
그는 말하지 않았다.
그녀도 더 이상 묻지 않았다.
```

(여백: 그의 반응은? 그녀의 표정은?)

---

### 기법 2: 디테일 대비

**디테일의 대비로 갈등을 만든다**

✅ **예시**:
```
그녀가 "괜찮아"라고 말할 때, 손가락이 잔 가장자리를 파고 있었다.
손끝이 하얗게 질릴 만큼.
```

(대비: 괜찮다고 말함 VS 손의 긴장)

---

### 기법 3: 반복의 힘

**핵심 정보를 적절히 반복해, 인상을 깊게 한다**

✅ **예시**:
```
그가 돌아올 거라고 했다.
그녀는 믿었다.

(몇 장 후)

그가 돌아올 거라고 했었다.
그녀는 기억하고 있다.

(또 몇 장 후)

그가 돌아올 거라고 했다.
하지만 오지 않았다.
```

## 총결

**Natural-Voice 스타일의 정수**:

1. **간단하고 직접적** - 가장 소박한 언어를 사용
2. **구체적으로 느낄 수 있게** - 보이고 만져지는 것을 쓴다
3. **대화가 왕** - 인물이 말하게 한다
4. **적을수록 많은 것** - 모든 장식을 삭제
5. **진실이 제일** - 사람이 이야기를 들려주는 것처럼

> 당신은 그저 이야기를 하나 들려주면 됩니다, 듣고 싶어하는 사람에게.
