# 문학 집필 스타일

## 스타일 포지셔닝

**핵심 이념**: 깊이, 사상성, 예술성

문학적 가치를 추구하며, 단순한 스토리성에 그치지 않습니다.

---

## 적용 시나리오

### ✅ 가장 적합
- 순수문학
- 순수 문학 창작
- 역사 소설 (현대 배경)
- 사실주의 소재
- 사상적 깊이가 필요한 작품

### ⚠️ 신중하게 사용
- 웹소설 쾌감소설 (리듬이 느려짐)
- 가볍고 오락 지향적 작품

## 핵심 원칙

### 1. 의경 우선

**사물을 묘사하는 것이 아니라, 감수를 묘사한다**

#### ❌ 직백적 묘사
```
밖에 비가 왔다, 그녀의 기분이 매우 안 좋았다.
```

#### ✅ 의경 묘사
```
커피 향기가 공기 속에서 천천히 퍼졌다. 임완청이 창가 자리에 앉아,
손안의 잔이 서서히 식어갔다, 지금 그녀의 마음처럼. 햇살이 이 오후를 잘라내고,
그녀는 아주 오래전 어느 순간을 떠올렸다 — 같은 빛, 같은 기다림,
단지 그때는 아직 기다림에 의미가 있다고 믿었을 뿐이다.
```

---

### 2. 절제된 서정

**감정이 있되, 범람하지 않는다**

#### ❌ 과도한 서정
```
그녀의 마음은 만 자루의 칼에 찔린 듯, 살고 싶지 않을 만큼 아프고,
눈물이 무너진 댐처럼 쏟아져 내렸다.
```

#### ✅ 절제된 서정
```
울고 싶었지만, 눈물이 나오지 않았다.
그냥 지쳤다, 아주 먼 길을 걸어온 것 같았다.
```

---

### 3. 철학적 사유와 인간성

**스토리를 통해 인간성을 탐구한다**

#### ✅ 예시
```
그를 미워하지 않았다. 정말로.

미움에는 힘이 필요한데, 그녀에게는 더 이상 남은 힘이 없었다.

마치 물에 빠진 사람이 물을 미워하지 않는 것처럼,
그저 필사적으로 떠오르려 할 뿐. 그런데 그녀에게는 떠오를 힘조차 없었다.

그래서 가라앉기를 택했다.

조용히, 낙엽 한 장처럼.
```

---

## 언어 특징

### 정확한 어휘

**모든 단어를 숙고한다**

#### ❌ 모호한 단어
```
매우 슬프고, 마음이 불편했다.
```

#### ✅ 정확한 단어
```
공허함을 느꼈다, 무언가가 빠져나간 것 같았다.
극심한 통증이 아니라, 무감각이었다.
```

---

### 적절한 비유

**비유는 신선해야 하고, 진부해서는 안 된다**

#### ❌ 진부한 표현
```
시간이 쏜살같이 흘렀다.
그녀의 사랑은 불나방 같았다.
```

#### ✅ 신선한 비유
```
시간은 무능한 도둑 같아서,
많은 것을 훔쳐갔지만, 가장 남기지 말아야 할 기억을 남겨두었다.

그녀의 사랑은 농담 같았다,
다만 그녀가 웃을 수 없었을 뿐.
```

---

### 문형의 리듬

**장단문 교차로 운율을 만든다**

#### ✅ 예시
```
오랫동안 걸었다. (단문)

매일 지나가던 그 거리를, 절대 들어가지 않을 그 카페를,
한때 자신을 안아주었던 그 모퉁이를 지나갔다. (장문)

오랫동안 걸었다. (반복 단문)

하지만 어디로 가야 할지 몰랐다. (중문)
```

**리듬 분석**:
- 단문으로 시작 (톤 설정)
- 장문으로 펼침 (상세)
- 단문 반복 (강조)
- 중문으로 마무리 (여백)

## 심리 묘사

### 심층 심리 묘사

**Literary 스타일은 깊은 심리 묘사를 허용한다**

#### ✅ 예시
```
가끔 생각했다, 그때 다른 길을 택했다면 어땠을까.

하지만 곧 그 생각을 부정했다. 자신의 선택이 옳았다고 생각해서가 아니라,
이미 존재하지 않는 가능성을 생각하는 것이 너무 지치기 때문이었다.

사람은 늘 앞으로 나아가야 한다.

앞에 무엇이 기다리고 있어서가 아니라,
앞으로 가지 않으면 제자리에 서서
그 기억들이 천천히 자신을 삼키는 것을 바라볼 수밖에 없으니까.
```

**요점**:
- 비교적 긴 심리 묘사 허용
- 철학적 사유가 있음
- 유수장이 아니라, 사고의 과정

---

### 의식의 흐름 기법

**적절히 의식의 흐름을 사용한다**

#### ✅ 예시
```
비. 또 비가 온다. 뭔가 떠올랐다. 3년 전? 아니 4년 전?
기억이 안 난다. 어차피 다 똑같다. 비 오는 날은 항상 떠올리면 안 되는 것들을
떠올리게 한다. 비가 싫다. 왜 싫은지는 잘 모르겠다.
어쩌면 그냥 비 오는 날엔 밖에 나갈 수 없어서. 어쩌면 아닐 수도.
어쩌면 이유 같은 건 원래 없을 수도.

어떤 일은, 이유가 필요 없다.

예를 들어 비를 싫어하는 것. 예를 들어 누군가를 그리워하는 것.
```

## 장면 묘사

### 환경과 감정의 융합

**환경은 배경이 아니라, 감정의 외화이다**

#### ❌ 단순한 환경 묘사
```
카페에 사람이 많지 않고, 인테리어가 정갈하고, 조명이 부드러웠다.
```

#### ✅ 환경의 감정화
```
카페에는 서너 명의 손님만 있었고, 각자 구석에 앉아,
마치 약속한 것처럼, 딱 맞는 거리를 유지하고 있었다.

그녀가 창가 자리에 앉으니, 유리에서 굴절된 빛이 들어와,
그녀의 그림자를 바닥에 드리웠다, 흐릿하게, 언제라도 사라질 것처럼.
```

---

### 디테일의 상징적 의미

**디테일에는 깊은 의미가 있어야 한다**

#### ✅ 예시
```
커피가 이미 식었다.

언제부터인지, 식은 커피를 마시는 습관이 들었다.
좋아해서가 아니라, 항상 뜨거울 때 마시는 것을 잊어버리니까.

그녀는 항상 잊었다, 사랑할 때 제대로 사랑하는 것도.
```

**상징**: 식은 커피 = 놓친 타이밍

## 대화 설계

### 여백이 있는 대화

**대화에는 서브텍스트가 있어야 한다**

#### ✅ 예시
```
"잘 지내?"

"잘 지내."

"진짜?"

"응."

그녀는 그가 거짓말하는 걸 알았다. 그도 그녀가 아는 걸 알았다.
하지만 둘 다 들추지 않았다.

어떤 거짓말은, 유지되어야 하니까.
```

**요점**:
- 대화가 짧음
- 서브텍스트가 풍부
- 독자에게 상상의 여지를 남김

---

### 철학적 대화

**대화를 통해 사상을 표현한다**

#### ✅ 예시
```
"영원을 믿어?"

"안 믿어."

"왜?"

"왜냐면," 잠시 멈추고, "영원의 전제는 아무것도 변하지 않는 것이거든.
하지만 이 세상에서, 유일하게 변하지 않는 것은 모든 것이 변한다는 것이야."

"그럼 사랑은?"

"사랑도 그래." 그가 그녀를 보며, "그래서 우리가 해야 할 것은,
사랑을 영원하게 만드는 게 아니라, 매 순간을 기억할 가치가 있게 만드는 것이야."
```

## 서사 기법

### 비선형 서사

**시간 순서를 뒤섞을 수 있다**

#### ✅ 예시
```
비 오는 어느 오후에 그를 만났다.

【시간 점프: 10년 전】

그날도 비가 왔다. 그를 처음 본 것은 도서관 앞이었다.
검은 우산을 들고 있었다...

【현재로 돌아옴】

10년이 지났다. 비는 여전히 그렇게 온다. 하지만 우산을 든 사람은, 이미 없다.
```

---

### 다중 시점

**다른 인물의 시점으로 전환할 수 있다**

#### ✅ 예시
```
【임완청의 시점】
분명 자기를 미워할 거라 생각했다.
안 그러면 왜 쳐다보지도 않겠어.

【심이행의 시점】
감히 쳐다볼 수가 없었다.
한번만 보면, 힘겹게 눌러놓은 감정이
전부 쏟아져 나올까 봐.
```

## 완전한 예시

### 장면: 재회

#### ✅ Literary 스타일 버전
```
비 오는 어느 오후에 그를 만났다.

10년이 지났다. 그의 얼굴을 이미 잊었다고 생각했는데,
카페 문 앞에 나타난 그 순간,
어떤 것은 뼛속에 새겨져 있다는 걸 알았다.

그도 그녀를 봤다.

그렇게 서로를 바라보았다, 아무도 먼저 입을 열지 않았다.
입을 열면, 이 10년의 시간이 정말로 과거가 될까 봐.

결국 그녀가 먼저 말했다.

"잘 지냈어?"

이렇게 진부한 인사. 웃고 싶었지만, 웃을 수가 없었다.

"잘 지냈어." 그의 목소리가 조금 쉬었다, "너는?"

"나도 잘 지냈어."

얼마나 우스운가. 10년 만에 만나 한다는 말이 고작 "잘 지냈어".

그런데 "잘 지내다"란 뭘까?

이 10년 동안, 수없이 한밤에 깨어나,
그의 모습을 떠올리고, 다 지나갈 거야,
괜찮아질 거야, 하고 자신에게 말했던,
그런 "잘 지내다"일까?

아니면 그가 이 10년 동안, 많은 곳을 가고 많은 일을 했지만,
항상 뭔가 부족한 느낌이었고, 오늘에서야
부족했던 것이 한 사람이었다는 걸 깨달은,
그런 "잘 지내다"일까?

비가 계속 내렸다.

창밖 세상이 흐릿하게 번졌다.

오직 그들만, 여전히 여기에, 선명하게 존재하고 있었다.
```

**분석**:
- 의경 조성 (비, 카페)
- 심리적 깊이 (10년, 잊음, 기억)
- 철학적 사유 ("잘 지내다"란 무엇인가)
- 절제된 감정 (웃고 싶지만 웃을 수 없다)
- 여백 (결말을 명시하지 않음)

## 총결

**Literary 스타일의 정수**:

1. **깊이가 있다** - 단순한 이야기가 아니라, 사상이 있어야
2. **절제가 있다** - 감정이 진실하되 범람하지 않아야
3. **미감이 있다** - 언어가 아름답되 화려하지 않아야
4. **인간성이 있다** - 인간성의 복잡함을 탐구
5. **여백이 있다** - 다 말하지 않고 독자가 생각하게

---

**기억하세요**:

> 문학은 기교 뽐내기가 아니라, 가장 적절한 문자로,
> 가장 진실한 감정과 가장 깊은 사유를 표현하는 것입니다.
