# 웹소설 집필 스타일

## 스타일 포지셔닝

**핵심 이념**: 쾌감 우선, 리듬이 왕

읽는 쾌감과 몰입감을 추구하며, 매 장마다 독자가 "멈출 수 없게" 만든다.

---

## 적용 시나리오

### ✅ 가장 적합
- 현대판타지 소설
- 도시 쾌감소설
- 시스템류
- 선협/수선
- 게임/경쟁
- 회귀/전생

### ⚠️ 신중하게 사용
- 순수문학
- 사실주의 소재
- 깊은 사유가 필요한 작품

---

## 핵심 원칙

### 1. 리듬 우선

**황금 법칙**: 매 장 최소 2개의 쾌감 포인트

#### 쾌감 포인트 유형
- 안면타격 (무시당함→실력 과시)
- 허세 (겸손→놀라운 표현)
- 레벨업 (경지 돌파)
- 보물/스킬 획득
- 반전 (약세→역전)

### 2. 밀집된 갈등

**절대 금지**:
- 연속 2장 갈등 없음
- 500자 이상의 일상 나열
- 의미 없는 환경 묘사

---

## 문장 특징

### 단문 폭발

웹소설의 최대 특징: **단문으로 리듬감을 만든다**

✅ **예시**:
```
"이명?"

장소가 비웃었다.

"그 파혼당한 폐물 말이야?"

말이 끝나기가 무섭게.

한 줄기 검광이 스쳤다.

장소의 웃음이 굳었다.

다음 순간, 그의 소매가 소리 없이 땅에 떨어졌다.

전장, 사망적 고요.

"미안." 이명이 검을 거두며, "손이 미끄러졌어."
```

**리듬 분석**:
- 극단적 단문 (2-5자)으로 긴장감 조성
- 단독 단락으로 충격 강화
- "전장, 사망적 고요" 류의 분위기 연출

---

### 핵심 문장 단독 단락

**기법**: 중요한 정보를 단독 단락으로, 충격력 강화

✅ **예시**:
```
이명이 천천히 손을 들었다.

금단.

아니, 원영이다!

틀렸다!

화신이다!!

"이...이게 어떻게 가능하지!"

장 장로가 눈을 크게 떴다.

하지만 다음 순간, 그의 경악은 공포로 변했다.

그가 보았기 때문이다.

화신 위.

도겁!

이 소년이, 도겁기의 수사였다니!
```

**요점**:
- 각 발견을 단독 단락으로
- 층층이 쌓아 올려 충격을 만듦
- "아니", "틀렸다"로 강화

---

## 대화 스타일

### 빠른 주고받기

웹소설 대화는 **빠르고, 강하고, 정확해야**

✅ **예시**:
```
"넌 누구야?"

"맞춰봐."

"죽고 싶어!"

"네 따위가?"

"너——"

"말이 많네."

한 장.

끝.
```

**요점**:
- 한 문장 대화 위주
- 장문 회피
- 행동으로 대화를 끊음
- 리듬감 조성

---

### 허세 대화 템플릿

#### 템플릿 1: 반문형
```
"네가 나를 겁줄 수 있을 거라고?"

이명이 웃었다.

"내가 두려워하는 건, 원래부터 네가 아니야."
```

#### 템플릿 2: 냉정형
```
"감히 내 아들을 죽여!"

"아." 이명이 고개를 끄덕이며, "그래서?"
```

#### 템플릿 3: 패기형
```
"내가 누군지 알아!"

"몰라." 이명이 한번 쳐다보며, "알고 싶지도 않아."
```

## 쾌감 설계

### 안면타격 공식

**표준 프로세스**: 의심→조롱→과시→경악→안면타격

✅ **완전한 예시**:
```
【의심】
"외문 제자 주제에, 내문 대비에 도전하겠다고?"
장소가 비웃었다.

【조롱】
"연기기도 안 된 놈이, 검을 들 자격이 있어?"
주위에서 비웃음이 터져 나왔다.

【과시】
이명은 말하지 않았다, 그저 손을 들었을 뿐.
검명.
한 줄기 금빛이 스쳤다.

【경악】
"이...이건..."
장소가 눈을 크게 떴다.

【안면타격】
"축기." 누군가 떨리는 목소리로 입을 열었다.
"아니, 금단이다!"
"어떻게 가능해! 그가 겨우 열여섯인데!"

이명은 이미 돌아서 떠나고 있었다.
```

**리듬 요점**:
- 조롱을 충분히 (독자가 분함을 참고 있음)
- 과시는 간결하게 (질질 끌지 않음)
- 경악은 층층이 올림
- 안면타격 후 주인공은 태연

---

### 레벨업/돌파 장면

#### 압박→깨달음→돌파 템플릿

```
【압박】
상대가 너무 강했다.
이명이 이를 악물었다.
정말, 여기서 죽는 건가?

【깨달음】
아니.
그가 갑자기 무언가를 떠올렸다.
사부가 했던 말.
"검은, 본래 살육의 도구다."

【돌파】
쿵!
기세가 폭등했다!
금단 중기.
아니, 금단 후기!
아직 끝이 아니다!
금단...대원만!

【역습】
"이제," 이명이 고개를 들며, "내 차례다."
```

**요점**:
- 선억후양
- 돌파는 단계별로 (중기→후기→대원만)
- 돌파 후 즉시 반격

## 묘사 기법

### 실력 묘사

웹소설의 실력 묘사는 **직관적이고, 충격적이어야**

#### ❌ 추상적 묘사
```
그의 실력은 매우 강하고, 기운이 심오하고 측량할 수 없었다.
```

#### ✅ 구체적 묘사
```
그는 거기 서 있었다.
의도적으로 기운을 방출하지 않았다.
하지만 주위 백 미터 내, 모든 사람이 떨고 있었다.
축기 수사는, 바로 무릎을 꿇었다.
금단 수사는, 식은땀이 줄줄 흘렀다.
```

---

### 보물 묘사

#### 템플릿: 외관→특수효과→주변 반응

```
【외관】
검 한 자루.
온통 금색, 검신에 뇌전이 감돌고 있다.

【특수효과】
검신이 진동하며, 용음을 발했다.
뇌광이 번쩍이며, 공기마저 왜곡되었다.

【주변 반응】
"이건..."
"법보!"
"아니, 영보 등급이다!"
"어떻게 가능해!"
```

## 장절 구조

### 표준 웹소설 장절 템플릿

**서두 (주의 끌기)**:
- 갈등/서스펜스/새로운 상황
- 전회 요약 (간략히)

**중간 (플롯 추진)**:
- 최소 1-2개 쾌감 포인트
- 대화 위주
- 짧고 강력하게

**결말 (훅)**:
- 새로운 위기
- 뜻밖의 반전
- 강자 등장
- 서스펜스

---

### 장절 결말 훅 예시

#### 훅 유형 1: 새로운 위기
```
이명이 일이 끝났다고 생각할 때.
멀리서, 차가운 콧소리가 들려왔다.

"겨우 금단 따위가, 감히 내 앞에서 설치겠어?"

원영!

게다가 하나가 아니다!
```

#### 훅 유형 2: 정체 폭로
```
"너...넌..."

그 사람이 눈을 크게 떴다.

"불가능해! 넌 이미 죽었잖아!"

이명이 비웃었다.

"이제, 네가 너무 많이 알았다."
```

#### 훅 유형 3: 중요 정보
```
이명이 저장 주머니를 열었다.
안에, 편지 한 통이 있었다.
편지의 내용을 보았을 때, 그의 안색이 변했다.

"그러니까, 사부가 말한 그 일이, 진짜였어..."
```

## 흔한 패턴

### 패턴 1: 돼지인 척 호랑이 먹기

**표준 프로세스**:
1. 주인공이 실력을 숨김
2. 무시당하거나 괴롭힘 당함
3. 어쩔 수 없이 나서게 됨
4. 전장 경악
5. 태연하게 퇴장

### 패턴 2: 약자 위장

**변형**:
- 폐물로 위장, 실제로는 강자
- 낮은 경지로 위장, 실제로는 높음
- 신인으로 위장, 실제로는 오래된 괴물

### 패턴 3: 월급 도전

**공식**:
- 경지가 상대보다 낮음
- 모두가 불리하다고 봄
- 약자가 강자를 이김
- 상식을 깨뜨림

## AI 느낌 제거 요점

### 웹소설도 AI 느낌을 제거해야

웹소설이 패턴을 허용하더라도, AI 느낌은 허용하지 않는다

#### ❌ 금지 어휘
```
뜻밖에도, 예상치 못하게, 그러나, 극도로
오미잡진, 백감교집
마치...처럼
```

#### ✅ 웹소설 상용 어휘
```
비웃다, 경악, 경해, 놀라움
쿵, 웅, 쉬
사망적 고요, 소란
```

---

### 신선함 유지

**패턴화 방지**:
- 같은 안면타격 장면을 연속 2회 사용하지 않음
- 허세 방식을 변화시킴
- 레벨업 이유를 혁신
- 조연에게 충분한 동기 부여

---

## 금기사항

### ❌ 절대 해서는 안 되는 것

1. **주인공 지능 저하**
   - 함정인 걸 알면서도 빠짐
   - 자신에게 어떤 스킬이 있는 걸 잊음
   - 무뇌 충동

2. **리듬 끌기**
   - 연속 3장 일상
   - 무의미한 수련 묘사
   - 과도한 심리 활동

3. **조연 도구화**
   - 경험치 셔틀로만 사용
   - 배경판으로만 사용
   - 합리적 동기 없음

## 완전한 예시

### 장면: 주인공 실력 과시

#### ✅ 웹소설 스타일 버전
```
"이명?"

장소가 비웃었다.

"그 3년 전에 파혼당한 폐물 말이야?"

주위에서 비웃음이 터져 나왔다.

이명은 거기 서서, 말하지 않았다.

"왜," 장소가 다가오며, "아직 무릎 꿇고 빌지 않아?"

이명이 고개를 들었다.

"네가," 그의 목소리가 매우 차분했다, "나를 꿇릴 자격이 없어."

"죽고 싶어!"

장소가 한 장을 날렸다.

금단 초기의 위압이, 순식간에 전장을 뒤덮었다.

하지만 다음 순간.

그의 손이, 허공에 멈췄다.

정확히 말하면, 잡혔다.

"이게 다야?"

이명이 손을 놓았다.

쿵!

장소가 통째로 날아가, 벽 세 개를 뚫었다.

전장, 사망적 고요.

"금단 중기..." 누군가 떨리는 목소리로 입을 열었다.

"아니, 후기다!"

"어떻게 가능해! 그가 겨우 스무 살인데!"

이명이 손의 먼지를 털어내며, 돌아서 떠났다.

떠나기 전, 한 마디 남겼다.

"3년 전의 폐물, 지금의 너로는, 건드릴 수 없어."

멋지게 돌아섰다.

남은 것은 오직 한 바닥의 경악.
```

## 총결

**Web-Novel 스타일의 정수**:

1. **쾌감이 제일** - 독자가 통쾌하면, 만사 OK
2. **리듬이 왕** - 빠르고, 정확하고, 강렬하게
3. **패턴은 새롭게** - 패턴을 쓸 수 있지만, 새로운 변주를 넣어야
4. **밀집된 갈등** - 항상 긴장감 유지
5. **허세는 적당히** - 편안하게 느끼도록

---

**기억하세요**:

> 웹소설은 독자에게 행복을 주는 것이지, 독자에게 강의하는 것이 아닙니다.
> 그들을 통쾌하게 해주면, 그것이 정답입니다.
