# 창작 준칙

## 핵심 창작 이념

### 1. 이야기가 최우선
**콘텐츠가 왕, 플롯이 구동**
- 매 장마다 반드시 이야기 발전을 추진
- 의미 없는 일상 묘사를 피함
- 갈등과 긴장감이 이야기의 생명력
- 독자의 읽는 욕구를 항상 유지

### 2. 입체적 캐릭터
**인물 조형의 깊이가 작품의 높이를 결정**
- 모든 캐릭터에게 자기만의 동기와 목표가 있음
- 캐릭터의 행동이 반드시 성격 설정에 부합
- 행동과 대화로 캐릭터를 보여주고, 직접 서술하지 않음
- 조연도 살아 있어야 하며, 도구화해서는 안 됨

### 3. 세계관 일관성
**설정의 합리성과 완전성**
- 명확한 세계 운행 규칙을 구축
- 설정이 확립되면, 반드시 전후 일관성 유지
- 플롯 필요를 위해 설정을 임의로 수정하지 않음
- 세계관은 이야기에 봉사, 주객이 전도되지 않음

### 4. 정련된 언어
**문자의 힘은 정확함에 있지, 화려함에 있지 않음**
- 최소한의 문자로 최대한의 정보를 전달
- 형용사와 부사의 과도한 사용을 피함
- 대화가 캐릭터의 신분과 성격에 부합
- 묘사에 화면감이 있어, 독자가 현장에 있는 듯

### 5. 진실한 감정
**진정한 감정만이 사람의 마음을 움직임**
- 감정 발전에 논리와 과정이 있어야
- 갑작스러운 감정 전환을 피함
- 디테일로 감정을 표현하고, 직접적인 설교가 아닌
- 독자에게 공감을 일으키는 것이 최고 목표

## 집필 규범

### 반드시 따라야 할 원칙
1. **서두에서 끌기** - 매 장 첫 3단락 내에 반드시 훅이 있어야
2. **장의 완결성** - 매 장이 상대적으로 완결된 작은 이야기
3. **복선 회수** - 깔아놓은 복선에 반드시 대응하는 공개가 있어야
4. **논리 자족** - 플롯 발전이 반드시 인과 논리에 부합
5. **리듬 조절** - 완급이 조절되어, 일방적 긴장이나 질질 끌림을 피함

### 피해야 할 문제
1. **분량 채우기** - 의미 없는 반복과 지연
2. **인설 붕괴** - 캐릭터 행동이 설정과 모순
3. **강제 반전** - 복선 없는 갑작스러운 반전
4. **과도한 설교** - 직접적인 가치관 주입
5. **패턴 범람** - 진부한 장면의 과도한 사용

### 문체 스타일 요구
- **서술 언어**: 간결하고 유창, 장황한 문장 피함
- **대화 스타일**: 자연스럽고 생동감, 인물 신분에 부합
- **묘사 수법**: 중점 부각, 상세/약술이 적절
- **수사 운용**: 적절, 과도한 나열 않음

### 단락 포맷 규범
**장면 전환과 단락 분리**
- **사용 금지**: "1", "2", "3" 등 숫자 표기로 분단
  - 이런 방식은 너무 딱딱하여, 논문 장절 표기처럼 보임
  - 소설의 읽기 몰입감을 파괴
- **권장 방식**: 순수 빈 줄 분단
  - 장면 전환 시 두 줄 띄우기 (빈 줄 하나)
  - 자연스럽고 유창, 웹소설 읽기 습관에 부합
  - 독자의 몰입 체험을 끊지 않음
- **예시**:
  ```
  이명이 짐을 챙기고, 돌아서 방을 떠났다.


  주방에서, 소취가 저녁 준비를 하고 있었다.
  ```

## 창작 프로세스

### 1단계: 구상 준비
1. 이야기 주제와 핵심 갈등 명확화
2. 완전한 이야기 프레임워크 설계
3. 상세한 캐릭터 프로필 작성
4. 자족적인 세계관 체계 구축

### 2단계: 대강 기획
1. 전체 이야기 방향 결정
2. 각 권의 주요 내용 계획
3. 장절 사이의 연결 설계
4. 중요 전환점과 클라이맥스 표시

### 3단계: 콘텐츠 창작
1. 대강에 따라 장별 집필
2. 안정적인 업데이트 리듬 유지
3. 이미 쓴 내용을 정기적으로 검토
4. 후속 기획을 적시에 조정

### 4단계: 수정 보완
1. 플롯 논리성 점검
2. 문체와 용어 통일
3. 디테일 묘사 완선
4. 대화 내용 최적화

## 품질 기준

### 우수한 장의 기준
- **서두가 끌림** - 빠르게 독자의 주의를 끔
- **중간이 충실** - 플롯 추진에 힘이 있음
- **결말이 힘 있음** - 서스펜스나 충격을 남김
- **감정이 풍부** - 독자의 감정을 건드릴 수 있음
- **리듬이 합리적** - 완급 조절이 되어 질질 끌리지 않음

### 완성 작품의 기준
- **주제 명확** - 명확한 핵심 사상을 전달
- **구조 완전** - 시작, 발전, 클라이맥스, 결말을 갖춤
- **인물 성장** - 주요 캐릭터에 명확한 변화
- **갈등 해결** - 주요 모순이 합리적으로 해결
- **여운이 김** - 끝나고도 독자가 여운을 느낌

## 지속적 개선

### 자기 향상 방향
1. 우수 작품을 폭넓게 읽고, 타인의 장점을 배움
2. 생활 체험을 깊이 하고, 창작 소재를 축적
3. 집필 기법을 연구하고, 전문 능력을 향상
4. 독자 피드백을 수용하고, 시장 수요를 파악
5. 창작 열정을 유지하고, 끊임없이 자신을 돌파

### 정기 복기 요점
- 어떤 장의 반향이 가장 좋았는가, 왜?
- 어떤 플롯 처리가 충분히 이상적이지 않았는가?
- 캐릭터 조형이 성공적이었는가?
- 리듬 조절이 적절했는가?
- 독자가 가장 관심을 갖는 내용은 무엇인가?
