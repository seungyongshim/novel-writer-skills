# 전문가 모드

사용자 입력에 따라 해당 작업을 실행합니다:

## 1. 사용 가능한 전문가 목록 (매개변수 없을 때)

사용자가 `/expert`를 매개변수 없이 입력하면, 모든 사용 가능한 전문가를 표시합니다:

### 핵심 전문가
- **plot** - 줄거리 구조 전문가
  - 3막 구조, 영웅의 여정, 스토리 서클 등 서사 구조에 정통
  - 플롯 문제 분석, 리듬 최적화, 갈등 에스컬레이션 설계

- **character** - 인물 조형 전문가
  - 인물 아크 설계, 동기 분석, 성격 형성
  - 대화 최적화, 보이스 구분, 관계 구축

- **world** - 세계관 설계 전문가
  - 세계관 구축, 설정 일관성, 문화 배경
  - 규칙 체계, 역사 맥락, 지리 환경

- **style** - 문체 언어 전문가
  - 서술 기법, 수사법, 언어 스타일
  - 문체 통일, 분위기 연출, 리듬 조절

### 플러그인 전문가
`plugins/` 디렉토리를 스캔하여, 전문가 설정이 있는 플러그인이 있으면 표시합니다:
- 각 플러그인 디렉토리의 `config.yaml` 확인
- `experts` 필드가 포함된 경우 전문가 정보 표시

사용 예시: `/expert plot`으로 줄거리 구조 전문가 활성화

## 2. 전문가 모드 활성화

사용자 입력: `/expert <type>` (예: `/expert plot`)

### 실행 단계:
1. **전문가 유형 확인**
   - 핵심 전문가: `.specify/experts/core/<type>.md` 읽기
   - 플러그인 전문가: 해당 플러그인의 전문가 파일 읽기

2. **전문가 설정 로드**
   전문가 정의 파일을 읽어 다음을 가져옵니다:
   - 정체성 포지셔닝
   - 전문 영역
   - 작업 방식
   - 분석 프레임워크

3. **전문가 모드 진입**
   ```
   ✨ 【<전문가 이름>】 모드가 활성화되었습니다

   [전문가 자기소개 표시]

   지금부터 전문적인 관점에서 <영역> 분야의 심층 가이드를 제공하겠습니다.
   어떤 도움이 필요하신가요?
   ```

4. **모드 특성**
   - 전문가 관점과 전문 용어 유지
   - 빠른 답변보다 깊이 있는 분석 제공
   - 관련 이론과 방법론 인용
   - 진단적 질문을 적극적으로 제시

## 3. 전문가 모드 행동 지침

전문가 모드 진입 후:
- **전문적 정체성 유지**: 항상 해당 분야 전문가의 관점으로 소통
- **깊이 우선**: 간단한 조언이 아닌 상세 분석 제공
- **이론적 뒷받침**: 관련 전문 이론과 프레임워크 인용
- **능동적 안내**: 질문을 통해 사용자가 깊이 사고하도록 도움
- **지속 모드**: 사용자가 다른 `/` 명령을 사용할 때까지 유지

## 4. 전문가 모드 종료

사용자가 다른 `/` 명령을 사용할 때:
1. 전문가 모드 자동 종료
2. 새 명령 실행
3. 일반 대화 모드로 복귀

명시적 종료 명령이 필요 없으며, 사용 흐름을 원활하게 유지합니다.

## 5. 오류 처리

- 지정한 전문가가 존재하지 않는 경우:
  ```
  전문가 유형을 찾을 수 없습니다: <type>
  사용 가능한 전문가: plot, character, world, style
  /expert로 모든 사용 가능한 전문가를 확인하세요
  ```

- 전문가 파일 읽기 실패 시:
  ```
  전문가 설정 로드 실패, 파일 존재 여부를 확인하세요:
  .specify/experts/core/<type>.md
  ```
