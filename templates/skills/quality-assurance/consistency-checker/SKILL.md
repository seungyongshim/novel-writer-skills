# 스토리 일관성 모니터링

## 자동 검사 시스템

### 본 스킬이 모니터링하는 것

#### 캐릭터 일관성

- **외적 특징**: 눈 색깔, 키, 나이, 흉터
- **성격**: 행동이 확립된 캐릭터에 부합하는지
- **지식**: 캐릭터가 알아야 할 것만 아는지
- **성장**: 변화가 캐릭터 아크와 일치하는지

#### 세계 규칙

- **마법/기술 시스템**: 능력의 사용이 일관적
- **지리**: 거리와 장소가 안정적
- **사회 규칙**: 문화와 관습이 모순되지 않음
- **물리 법칙**: 확립된 규칙이 무작위로 깨지지 않음

#### 타임라인 논리

- **사건 순서**: A가 논리적으로 B 전에 발생
- **시간 경과**: 캐릭터가 적절히 나이 들기
- **동시 사건**: 다중 POV 타임라인 정렬
- **역사 일관성**: 과거 참조가 일관적

### 작동 방식

**수동 모니터링**: 글을 쓰거나 이야기를 논의할 때, 자동으로 다음을 교차 참조합니다:
1. `characters/` 디렉토리의 캐릭터 프로필
2. `worldbuilding/` 디렉토리의 세계관 문서
3. `spec/tracking/timeline.json`의 타임라인 데이터
4. 이전 챕터의 내용

**어떤 행동도 필요 없습니다** - 모니터링은 백그라운드에서 진행됩니다.

## 문제가 감지되면

### 경고 형식

잠재적 불일치를 감지하면, 다음과 같이 알립니다:

**⚠️ 일관성 검사 경고**

```
문제: 캐릭터 특징 불일치
위치: 현재 챕터, 3번째 단락
참조: characters/mary-chen.md, 15행

현재 텍스트: "마리의 초록색 눈이 가늘어졌다..."
확립된 특징: "눈 색깔: 파란색" (3장에서 설정)

가능한 해결 방안:
1. 현재 텍스트를 "파란색 눈"으로 변경
2. 설정을 변경하려면, 캐릭터 프로필 업데이트
3. 비슷한 이름의 다른 캐릭터인가요?

자동 수정을 원하시나요, 직접 처리하시겠습니까?
```

### 심각도 수준

| 수준 | 아이콘 | 행동 | 예시 |
|------|-------|------|------|
| **치명적** | 🔴 | 즉시 중지하고 수정 | 캐릭터가 알 수 없는 비밀 정보를 갑자기 앎 |
| **경고** | ⚠️ | 빠른 시일 내 수정 | 캐릭터의 일상적 말투 패턴이 변함 |
| **참고** | 📝 | 확인 고려 | 타임라인이 압축된 느낌 |

## Novel-Writer 명령과의 통합

### `/write` 중

- 콘텐츠 생성 시 실시간 일관성 검사
- 치명적 문제에 대한 즉각 경고
- 규격 문서 자동 참조

### `/analyze` 중

- 포괄적 일관성 보고서
- 누적된 모든 경고와 참고사항
- 심각도순으로 정렬된 수정 제안

### `/track` 중

- 검증된 정보로 추적 데이터 업데이트
- 수동 검토를 위한 불일치 표시
- 일관성 이력 유지

## 설정

### 엄격도 수준

일관성 검사의 엄격도를 조정할 수 있습니다:

**엄격 모드** (비판타지 기본):
- 모든 모순 표시
- 현실 세계 물리 적용
- 타임라인이 완전히 논리적이어야

**유연 모드** (판타지/SF 추천):
- "멋진 규칙" 예외 허용
- 마법/기술이 현실을 구부릴 수 있음
- 예술적 허용을 인정하되 알림

**최소 모드**:
- 치명적 모순만 표시
- 캐릭터와 주요 플롯 포인트에 집중
- 사소한 불일치는 통과

### 특정 검사 비활성화

특정 불일치가 의도적인 경우:
```
"꿈 시퀀스에 대한 타임라인 검사를 비활성화해 주세요 -
의도적으로 비선형입니다."
```

## 흔한 오탐

때때로 실제로는 올바른 것을 표시할 수 있습니다:

### 의도적 모순

**예시**: 캐릭터가 눈 색깔을 거짓으로 말함
**수정**: 캐릭터 프로필에 주석 추가: "// 실제로는 파란색, 사람들에게 초록색이라고 말함"

### 믿을 수 없는 화자

**예시**: 1인칭 화자가 사건을 잘못 기억
**수정**: 헌법에 명시: "신뢰할 수 없는 화자 - 기억의 불일치는 의도적"

### 시간 점프

**예시**: 캐릭터의 나이가 갑자기 증가
**수정**: 챕터에서 시간 점프를 명확히: "3년 후..."

## 모범 사례

### 참조 문서를 최신 상태로 유지

일관성 검사기는 문서만큼만 정확합니다:
- 특징이 변경되면 캐릭터 프로필 업데이트
- 세계 규칙을 명확히 문서화
- `/timeline` 명령으로 타임라인 파일 유지

### 경고를 적시에 처리

일관성 문제를 쌓이게 하지 마세요:
- 치명적 경고는 즉시 수정
- 집필 세션 종료 시 경고 검토
- 수정 단계에서 참고사항 일괄 처리

### `/track`과 함께 사용

일관성 검사 + 추적 시스템 = 강력한 조합:
- `/track --check`으로 심층 일관성 검증 실행
- `/track --fix`로 간단한 문제 자동 수정 가능
- 정기적으로 둘 다 사용 (매 5-10장)

## 검사 카테고리 상세

### 캐릭터 일관성 검사

**외적 묘사**:
```
✓ 검사: 키, 체중, 나이, 머리색, 눈 색깔
✓ 검사: 흉터, 문신, 고유 표시
✓ 검사: 복장 스타일, 액세서리
```

**행동 패턴**:
```
✓ 검사: 말투 패턴이 일관적인지
✓ 검사: 반응이 성격에 부합하는지
✓ 검사: 결정이 가치관에 부합하는지
✓ 검사: 기술/능력이 일관적인지
```

**지식 상태**:
```
✓ 검사: 캐릭터가 언제 무엇을 알게 되었는지
✓ 검사: 알아서는 안 되는 비밀을 모르는지
✓ 검사: 기억이 확립된 사실과 일치하는지
```

### 세계 규칙 검사

**마법/기술 시스템**:
```
✓ 검사: 능력이 확립된 한계 내에서 사용되는지
✓ 검사: 대가/비용이 일관되게 적용되는지
✓ 검사: 규칙이 플롯 편의를 위해 변경되지 않는지
✓ 검사: 예외에 사전 암시가 있는지
```

**지리와 거리**:
```
✓ 검사: 장소가 지도에서 일관적인지
✓ 검사: 이동 시간이 합리적인지
✓ 검사: 기후가 지리와 일치하는지
✓ 검사: 랜드마크가 이동하지 않는지
```

**사회와 문화**:
```
✓ 검사: 문화 규범이 일관적인지
✓ 검사: 언어와 방언이 안정적인지
✓ 검사: 사회 구조가 무작위로 변하지 않는지
✓ 검사: 종교/신앙이 일관적인지
```

### 타임라인 검사

**사건 순서**:
```
✓ 검사: 원인이 결과 전에 발생하는지
✓ 검사: 캐릭터가 있을 수 없는 곳에 있지 않은지
✓ 검사: 사건이 합리적 시간 범위 내에 발생하는지
```

**시간 경과**:
```
✓ 검사: 캐릭터가 적절히 나이 드는지
✓ 검사: 계절이 순서대로 변하는지
✓ 검사: 임신/회복에 적절한 시간이 소요되는지
✓ 검사: 기술 습득에 연습 시간이 필요한지
```

**다중 POV 동기화**:
```
✓ 검사: 동시 사건이 다른 POV에서 일치하는지
✓ 검사: 시간 점프가 POV 간에 정렬되는지
✓ 검사: 어떤 POV도 다른 POV의 미래를 알지 못하는지
```

## 자동 수정 기능

일부 간단한 문제에 대해, 자동 수정을 제공할 수 있습니다:

### 자동 수정 유형

**철자 변형**:
```
감지: 캐릭터 이름 철자가 불일치
제안: 가장 흔한 철자로 표준화
행동: 전역 찾기 및 바꾸기 (승인 후)
```

**숫자 불일치**:
```
감지: 캐릭터 나이가 챕터 간 불일치
제안: 타임라인 기반으로 올바른 나이 계산
행동: 올바른 숫자로 업데이트
```

**타임라인 충돌**:
```
감지: 사건 날짜가 확립된 타임라인과 충돌
제안: 알려진 순서에 맞게 날짜 조정
행동: 타임라인 참조 업데이트
```

## 보고서와 추적

### 일관성 보고서

정기적으로 (또는 요청 시) 생성합니다:

```markdown
## 일관성 보고서 - [날짜]

### 챕터 범위: 1-15장

### 감지된 문제

#### 치명적 (반드시 수정)
1. 12장: 캐릭터가 14장에서야 밝혀지는 정보를 알고 있음
   - 수정: 12장 장면을 재작성하거나 14장 공개를 앞당김

#### 경고 (수정 권장)
1. 8장: 캐릭터의 눈 색깔이 파란색에서 초록색으로 변함
   - 수정 제안: 8장을 파란색으로 업데이트하거나 캐릭터 프로필 수정
   
2. 10장: 수도에서 국경까지 이동이 1일만에 (이전 3일로 확립)
   - 수정 제안: 시간 점프 추가 또는 일수 조정

#### 참고 (확인 고려)
1. 5장: 캐릭터 반응이 특성에 맞지 않는 것 같음
   - 검토: 의도적 성장인지 불일치인지?

### 통계
- 총 검사: 456건
- 문제 발견: 8건
- 자동 수정: 3건
- 검토 필요: 5건

### 일관성 점수: 94%
```

### 추적 이력

발견되고 수정된 일관성 문제의 이력을 유지합니다:
- 패턴 식별에 도움
- 반복 실수 방지
- 시간에 따른 개선 표시
- 수정 시 유용

## 다른 스킬과의 협력

### Writing Techniques Skills과 배합

**대화 일관성**:
- 캐릭터 목소리가 유지되는지 검사
- 말투 패턴 검증
- 특성에 맞지 않는 대화 표시

**장면 일관성**:
- 설정 디테일 검증
- 물리적 가능성 검사
- 시간 경과 확인

### Genre Knowledge Skills과 배합

**장르 관습 일관성**:
- 장르 규칙이 일관되게 적용되는지 확인
- 함정이 일관되게 피해지는지 검사
- 리듬 패턴 검증

---

**기억하세요**: 일관성은 완벽함이 아닙니다 - 독자의 신뢰입니다. 세계 규칙이 신뢰할 수 있을 때, 독자는 몰입합니다. 규칙이 임의로 변하는 것처럼 보이면, 독자는 이탈합니다. 이 신뢰를 유지하도록 도와드립니다.

**최종 결정권은 항상 당신에게 있습니다** - 불일치가 예술적 선택이라면, 알려주시면 더 이상 표시하지 않겠습니다.
