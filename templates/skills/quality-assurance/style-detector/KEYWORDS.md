# 글쓰기 스타일 키워드 상세 목록

본 문서에는 모든 탐지 가능한 스타일의 상세 키워드 목록, 트리거 시나리오 예시와 활성화 후 동작 템플릿이 포함되어 있습니다.

---

## 1. natural-voice (자연스러운 목소리 스타일)

### 키워드 (하나만 있어도 트리거)
```
구어체, 생활적, 진실감, 현실감 있는
말하듯이, 문학적 어투 빼고, 서면어 빼고
자연스러운, 꾸미지 않는, 구어, 일상 대화
진실된, 생활에 가까운, 인위적이지 않은
```

### 트리거 시나리오 예시
> 사용자: "구어체 느낌으로 쓰고 싶어요, 너무 문학적이지 않게"
> 사용자: "진짜 같이 써주세요, 평소에 말하는 것처럼"

### 활성화 후 동작 템플릿
```markdown
🎨 스타일 탐지기 활성화

【구어체, 진실감】 등의 키워드를 감지했습니다.

제안 사용: **natural-voice** (자연스러운 목소리 스타일)

이 스타일의 특징:
- 구어체 표현 위주, 친구에게 이야기하듯이
- 생활 디테일이 풍부, 몰입감 강함
- 대화로 플롯 추진 (비율 40-60%)
- 문학적 어투와 서면어 회피

이 스타일이 특히 적합한 경우:
✅ 도시 현대소설
✅ 현실 소재
✅ 감정류 소설

상세 스타일 가이드를 보시겠습니까?
```

## 2. literary (문학 스타일)

### 키워드
```
문학성, 아름다운, 깊이 있는
순수문학, 의경, 시적 (현대문)
철학적, 인간 본성, 사상성, 문예적
내용이 있는, 깊이 있는, 피상적이지 않은
```

### 트리거 시나리오 예시
> 사용자: "순수문학을 쓰고 싶어요, 깊이 있는 것으로"
> 사용자: "문학성이 있어야 해요, 웹소설 같은 건 아니고"

### 활성화 후 동작 템플릿
```markdown
🎨 스타일 탐지기 활성화

감지됨: 【순수문학, 깊이 있는】

제안 사용: **literary** (문학 스타일)

이 스타일의 특징:
- 아름다움, 깊이, 문학성
- 의경과 철학적 사유 중시
- 적절한 심리 묘사
- 절제되지만 힘 있는 언어

이 스타일이 특히 적합한 경우:
✅ 순수문학 작품
✅ 역사소설 (현대 배경)
✅ 사상적 깊이가 필요한 이야기

상세 스타일 가이드를 보시겠습니까?
```

## 3. web-novel (웹소설 스타일)

### 키워드
```
웹소설, 쾌감물, 빠른 리듬, 패턴
사이다, 역전, 레벨업물, 무능력자물
판타지 스타일, 선협 스타일, 시스템물
쾌감 포인트, 갈등 밀집
```

### 트리거 시나리오 예시
> 사용자: "그냥 웹소설 느낌이요"
> 사용자: "리듬 빠르게, 쾌감물 같은"

### 활성화 후 동작 템플릿
```markdown
🎨 스타일 탐지기 활성화

감지됨: 【웹소설, 쾌감물, 빠른 리듬】

제안 사용: **web-novel** (웹소설 스타일)

이 스타일의 특징:
- 빠른 리듬, 갈등 밀집
- 빈번한 쾌감 포인트 (사이다, 역전, 레벨업)
- 짧고 힘 있는 대화
- 명확한 패턴이되 진부하지 않음

이 스타일이 특히 적합한 경우:
✅ 판타지소설
✅ 도시 쾌감물
✅ 시스템물, 회귀물

상세 스타일 가이드를 보시겠습니까?
```

## 4. ancient-style (고풍)

### 키워드
```
고풍, 고운, 고전문 (너무 무겁지 않게)
시적, 이미지, 여백, 무협풍
고전미, 전통적, 강호 느낌
고운이 있는, 고대 느낌, 고전적
```

### 트리거 시나리오 예시
> 사용자: "고풍 느낌의 글이 좋겠어요"
> 사용자: "무협소설 같은 분위기로"

### 활성화 후 동작 템플릿
```markdown
🎨 스타일 탐지기 활성화

감지됨: 【고풍, 무협풍】

제안 사용: **ancient-style** (고풍)

이 스타일의 특징:
- 고운이 있되, 순수 고전문은 아님
- 풍부한 이미지, 여백의 미학
- 시적인 묘사
- 현대적 어휘 회피

이 스타일이 특히 적합한 경우:
✅ 무협소설
✅ 선협 (문체 측면)
✅ 고대 배경 소설

상세 스타일 가이드를 보시겠습니까?
```

## 5. minimal (미니멀 스타일)

### 키워드
```
미니멀, 간결, 절제, 헤밍웨이식
하드보일드, 군더더기 없이, 장황하지 않게, 여백
설명하지 않는, 하드보일드 스타일, 심플
```

### 트리거 시나리오 예시
> 사용자: "간결하게, 묘사 너무 많지 않게"
> 사용자: "헤밍웨이 같은 스타일로"

### 활성화 후 동작 템플릿
```markdown
🎨 스타일 탐지기 활성화

감지됨: 【미니멀, 헤밍웨이식】

제안 사용: **minimal** (미니멀 스타일)

이 스타일의 특징:
- 고도로 간결, 모든 수식 제거
- 짧은 문장 위주, 경쾌한 리듬
- 여백, 설명하지 않음
- 헤밍웨이식 절제

⚠️ 경고: 이 스타일은 매우 극단적이어서, 모든 사람에게 적합하지 않습니다

이 스타일이 특히 적합한 경우:
✅ 추리 미스터리
✅ 하드보일드 소설
✅ 실험적 작품

상세 스타일 가이드를 보시겠습니까?
```

## 사용 설명

사용자가 위의 어떤 키워드를 언급하면, 해당하는 활성화 후 동작 템플릿에 따라 사용자와 상호작용합니다:

1. 탐지된 키워드 확인
2. 해당 스타일 제안
3. 특징과 적용 시나리오 간략 설명
4. 상세 가이드 열람 여부 문의

사용자가 확인하면, 지식 베이스 로드 및 설정 작성 단계를 계속 진행합니다.
