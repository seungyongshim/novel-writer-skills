# 자연스러운 대화 집필 기법

## 핵심 원칙

### 언어로 캐릭터를 보여주기

각 캐릭터는 고유한 목소리를 가져야 합니다:

1. **어휘 선택**
   - 교양 있는 vs 캐주얼
   - 격식체 vs 속어
   - 전문 용어 vs 일상 언어
   - 캐릭터 배경이 어휘를 결정

2. **문장 구조**
   - 짧고/끊기는 vs 길고/유려한
   - 완전한 문장 vs 단편
   - 단순 구조 vs 복잡한 종속절
   - 사고방식을 반영

3. **말투 패턴**
   - 끼어들기, 쉼, 반복
   - 입버릇과 군말
   - 독특한 표현 습관
   - 문화적/지역적 특색

4. **말하지 않는 것**
   - 회피하는 화제
   - 거짓말 패턴
   - 침묵의 순간
   - 말하지 않은 의미

## 서브텍스트가 직접적 표현보다 낫다

캐릭터의 뜻 vs 말하는 것:

### ❌ 직접적 대화 (설교식)

```
"네가 어젯밤에 어디 갔는지 거짓말해서 화가 나."
```

### ✅ 서브텍스트가 풍부한 대화

```
"출장 회의 어땠어?"
"좋았어."
"그랬겠지."
```

**왜 더 좋은가**:
- 독자가 추리에 참여
- 긴장감과 불편함 창출
- 더 현실적 (사람들은 직접적 충돌을 회피)
- 캐릭터 역학을 보여줌

### 서브텍스트의 레이어

**표면 레이어**: 글자 그대로의 의미
**감정 레이어**: 진짜 감정
**관계 레이어**: 권력과 친밀도
**주제 레이어**: 더 큰 이야기의 의미

## 끼어들기와 겹침

현실 대화는 질서정연하지 않습니다:

### 끼어들기 사용

```
"들어봐, 네가 생각하기에—"
"네가 뭘 생각하는지 모르잖아."
"—근데 내가 끝까지 말하게 해주면—"
쾅, 저 멀리 문이 닫혔다.
"됐어." 그녀는 돌아섰다.
```

**끼어들기를 쓸 때**:
- 감정이 고조될 때
- 급박하거나 공황일 때
- 권력 다툼
- 관계 역학을 보여줄 때

### 생각이 말을 끊기

```
"그냥 말하고 싶었는데—" 그의 얼굴에 뭔가가 스쳤다. "아니야. 별거 아니야."
```

**효과**:
- 내면의 갈등을 보여줌
- 신비감 창출
- 숨겨진 정보 암시
- 캐릭터의 자기 검열

## 흔한 실수

### ❌ 정보 쏟아붓기

**문제**: 캐릭터가 독자에게 정보를 전달하기 위해서만 말함

**나쁜 예**:
```
"아시다시피, 밥, 우리는 2015년 고등학교 때부터 친구인데,
그때 둘 다 농구부에 들어갔고, 그다음에 같이 대학에 가서
거기서 공학을 공부했지..."
```

**좋은 예**:
```
"3학년 때 그 버저비터 기억나?"
밥이 웃었다. "코치가 아직도 그 얘기 해."
```

**해결법**:
- 행동 속에서 정보 공개
- 캐릭터가 실제로 할 말만
- 완전한 역사 대신 짧은 암시
- 독자가 배경을 조합하게

### ❌ 모두가 같은 소리

**문제**: 모든 캐릭터가 같은 어휘와 말투 사용

**해결법**: 주요 캐릭터마다 "목소리 시트" 작성

**목소리 시트 예시**:

| 캐릭터 | 문장 길이 | 어휘 | 버릇 | 회피하는 것 |
|--------|---------|------|------|-----------|
| 장 의사 | 중장, 복잡 | 격식, 의학 용어 | 과잉 설명 | 속어 |
| 이 학생 | 짧, 단편 | 캐주얼, 인터넷 용어 | "그니까", "알지" | 모른다고 인정 |
| 왕 사장 | 간결, 명령형 | 비즈니스, 직접적 | 말을 아낌 | 자기 설명 |

### ❌ 캐주얼 대화의 완벽한 문법

**너무 격식적**:
```
"저는 가게에 가려고 합니다. 무엇을 사다 드릴까요?"
```

**자연스럽게**:
```
"가게 가는데, 뭐 사다 줄까?"
```

**구어체 기법**:
- 축약 (하려고 → 할라고, 가려고 → 갈래)
- 단어 생략 (나는 가게에 간다 → 가게 간다)
- 단편 문장
- 군말 (음, 어, 그게)

### ❌ 대화로 서술하기

**문제**: 캐릭터가 서술이어야 할 내용을 말함

**나쁜 예**:
```
"나는 일어나서, 문 쪽으로 걸어가서, 열었어. 배달부였어."
```

**이것은 대화가 아닙니다**:
- 사람들은 자기 행동을 서술하지 않음
- 작가의 게으름
- 실제 서술이나 보여주기를 사용

## 고급 기법

### 대화를 행동으로

대사 태그로 캐릭터 상태를 보여줌:

```
"됐어." 그녀가 중얼거렸다. (좌절)
"됐어!" 그녀가 날카롭게 말했다. (분노)
"됐어..." 그녀의 목소리가 사그라들었다. (불확실)
```

**"말했다" 넘어서**:
- 속삭이다, 중얼거리다, 으르렁대다 (음량)
- 날카롭게 말하다, 외치다, 고함치다 (강도)
- 비꼬다, 조롱하다, 나직이 말하다 (어조)
- 하지만 남용 금지 - "말했다"로 보통 충분

### 침묵을 대화로

때로 말하지 않는 것이 중요합니다:

```
"나를 사랑해?"
그는 자기 신발을 내려다보았다.
```

**침묵의 힘**:
- 말로 표현할 수 없는 것을 전달
- 긴장감 창출
- 불편함이나 고통을 보여줌
- 독자가 채우게 함

### 행동으로 대화 끊기

캐릭터를 말하는 머리로 만들지 않기:

```
"이건 내가 원하는 게 아니야." 그녀가 컵을 밀어냈다. "이런 식으로는 안 돼."

그가 방을 서성였다. "그럼 뭘 원하는데?"

"나는—" 그녀의 손이 테이블 가장자리를 움켜쥐었다. "모르겠어."
```

**효과**:
- 단조로움 타파
- 바디랭귀지 보여줌
- 시각적 요소 추가
- 리듬 변화 창출

## 대화의 목적

### 모든 대화는 최소 다음 중 하나를 해야:

1. **캐릭터 드러내기**
   - 성격 보여주기
   - 동기 드러내기
   - 관계 보여주기
   - 성장 나타내기

2. **플롯 추진**
   - 중요 정보 제공
   - 결정 내리기
   - 갈등 만들기
   - 문제 해결하기

3. **분위기 구축**
   - 톤 설정
   - 긴장감 만들기
   - 유머 제공
   - 감정 심화

4. **갈등 보여주기**
   - 목표 대립
   - 오해
   - 권력 다툼
   - 숨겨진 의도

**대화가 이것들을 안 한다면**: 삭제하세요

## 특수 장면의 대화

### 다툼/갈등

**효과적 기법**:
- 짧은 문장, 빠른 왕복
- 잦은 끼어들기
- 말이 점점 날카로워짐
- 후회할 말을 할 수도
- 상승 후 잦아들기 (또는 폭발)

```
"넌 항상 그래."
"그래 어떤데?"
"가장하는 거—"
"나는 아무것도 가장 안 해!"
"—네가 신경 쓰는 척하는 거!"
```

### 로맨틱/친밀

**효과적 기법**:
- 부드러운 어조
- 불완전한 문장 (감정)
- 물리적 근접 묘사
- 풍부한 서브텍스트
- 취약함

```
"그냥..." 그의 엄지가 그녀의 턱을 스쳤다. "이걸 망치고 싶지 않아서."

"그러면 망치지 마." 그녀가 미소 지었다. "간단하잖아."

"간단하지." 그가 웃었다. "그래."
```

### 서스펜스/긴장

**효과적 기법**:
- 속삭임이나 짧은 말
- 불완전한 생각
- 끼어듦 (외부 위협)
- 의미를 담은 침묵
- 말할 수 없는 두려움

```
"저거 들었어?"

적막. 그리고: 멀리서 발자국 소리.

"우리 빨리—"

나뭇가지가 부러졌다. 둘 다 얼어붙었다.
```

## Novel-Writer 명령과의 통합

### `/specify` 시
- 주요 캐릭터의 고유한 목소리 정의
- 핵심 대화 장면 식별
- 주로 대화를 통해 드러낼 것 계획

### `/plan` 중
- 높은 긴장도 대화 장면 배치
- 대화를 통한 정보 공개 계획
- 캐릭터 목소리 아크 설계 (말하는 방식이 어떻게 변하는지)

### `/write` 시
- 캐릭터 목소리 일관성 자동 적용
- 정보 쏟아붓기 검사
- 서브텍스트 기회 제안
- 캐릭터 프로필로 대화 검증

### `/analyze` 중
- 캐릭터 목소리 일관성 검사
- 직접적/설교식 대화 식별
- 모든 대화에 목적이 있는지 검증
- 강화할 수 있는 영역 제안

## 대화 집필 체크리스트

- [ ] 각 캐릭터가 고유한 목소리를 가짐
- [ ] 대화가 플롯을 추진하거나 캐릭터를 드러냄
- [ ] 직접적 표현 대신 서브텍스트 사용
- [ ] 자연스러운 끼어들기와 겹침 포함
- [ ] 캐주얼한 말이 캐주얼하게 들림
- [ ] 정보 쏟아붓기 없음
- [ ] 행동이 큰 대화 덩어리를 끊음
- [ ] 적절한 곳에 침묵 사용
- [ ] 대사 태그가 다양하되 주의를 분산하지 않음
- [ ] 모든 대화에 명확한 목적

## 수정 팁

**소리 내어 읽기**:
- 자연스럽게 들리나?
- 발음하기 어려운 곳이 있나?
- 리듬이 유려한가?

**가리기 테스트**:
- 대사 태그를 가리고
- 대화만으로 누가 말하는지 구분할 수 있나?
- 못 한다면, 캐릭터 목소리가 충분히 독특하지 않음

**목적 테스트**:
- 이 대화가 어떤 목적을 달성하나?
- 없으면 뭘 잃나?
- 더 짧고 날카롭게 할 수 있나?

---

**기억하세요**: 훌륭한 대화는 자연스럽게 느껴지지만 정교하게 설계되었습니다. 말하는 것보다 더 많이 드러내고, 진실성을 유지하면서 이야기를 추진하며, 모든 단어에 목적이 있습니다. 적은 것이 더 많은 것입니다 - 핵심만 남기세요.
