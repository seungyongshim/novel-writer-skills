# 판타지소설 세계관 구축 규범

## 빠른 참조

| 요소 | 가이드 원칙 | 핵심 포인트 |
|------|-----------|-----------|
| **마법 시스템** | 명확한 규칙이 있어야 | 제한이 능력보다 중요 |
| **세계 설정** | 내적 일관성 | 모든 규칙에 이유가 있어야 |
| **종족/생물** | 독특하고 논리적 | 단순한 인간 변형 피하기 |
| **역사 깊이** | 최소 3세대 역사 | 과거가 현재에 영향 |
| **정치 구조** | 권력 분포 명확 | 갈등에 뿌리가 있어야 |

## 핵심 원칙

### Sanderson의 마법 법칙

**제1법칙**: 독자가 마법에서 얻는 만족감은, 마법을 이해하는 정도에 비례한다
- 마법으로 문제를 풀려면, 독자가 규칙을 이해해야
- 소프트 마법 시스템 (신비로운)은 분위기와 경이에 사용
- 하드 마법 시스템 (규칙 명확)은 문제 해결에 사용

**제2법칙**: 제한이 능력보다 더 흥미롭다
- 강력한 마법은 강력한 대가가 필요
- 제한이 갈등과 전략을 만듦
- 완벽한 시스템은 드라마를 만들 수 없음

**제3법칙**: 새것을 추가하기 전에, 기존 것을 먼저 확장하라
- 기존 요소 심화가 새 요소 추가보다 나음
- 상호 연결된 시스템이 고립된 것보다 강력
- 복잡성은 유기적으로 성장해야

## 마법 시스템 설계

### 하드 마법 시스템 (규칙 명확)

**필수 요소**:
1. **에너지 원천**
   - 마법은 어디서 오는가?
   - 유한한가 무한한가?
   - 고갈될 수 있는가?

2. **사용 규칙**
   - 누가 사용할 수 있나? 어떻게 획득하나?
   - 어떤 조건이 필요한가?
   - 어떤 제한이 있나?

3. **대가/결과**
   - 마법 사용의 비용은?
   - 과다 사용하면 어떻게 되나?
   - 장기적 영향이 있나?

4. **가능과 불가능**
   - 마법으로 무엇을 할 수 있나?
   - 명확히 할 수 없는 것은?
   - 경계는 어디인가?

**프레임워크 예시**:

```
마법 시스템: 원소 조종
원천: 모든 사람은 태어날 때 하나의 원소 친화력을 보유
규칙: 자기 원소만 조종 가능, 주위에 해당 원소가 존재해야
제한: 정신적 피로, 과다 사용 시 원소 역습
대가: 마법 사용 시 생명력 소모, 휴식으로 회복 필요
금기: 원소를 생성할 수 없고 조종만 가능; 생물 체내의 원소 조종 불가
```

### 소프트 마법 시스템 (신비감)

**특징**:
- 규칙이 독자에게 완전히 알려지지 않음
- 분위기와 경이에 더 많이 사용
- 주요 갈등을 편리하게 해결하면 안 됨
- 신비감과 경외감 유지

**사용 장면**:
- 고대 마법, 부분적으로 실전된
- 신의 힘, 범인의 이해를 초월
- 배경 경이와 세계 구축
- 메인라인 해결에 영향을 주지 않는 보조 플롯

## 세계관 구축 프레임워크

### 지리와 환경

**반드시 고려해야 할 것**:
1. **지형**
   - 산맥, 강, 해양이 문명에 어떤 영향?
   - 기후가 문화를 어떻게 형성하나?
   - 자원 분포가 경제에 어떤 영향?

2. **생태계**
   - 먹이사슬은 어떤 모습인가?
   - 마법 생물은 환경에 어떻게 적응하나?
   - 인간/기타 지적 종족은 어떻게 생존하나?

3. **마법이 지리에 미치는 영향**
   - 마법이 자연을 어떻게 바꾸나?
   - 마법으로 만들어진 랜드마크가 있나?
   - 마법 재난이 어떤 흔적을 남겼나?

### 사회와 문화

**계층 구조**:

**정치 차원**:
- 정부 형태 (군주제, 민주주의, 신정 등)
- 권력이 어떻게 전달되나?
- 마법이 정치에 어떤 영향?
- 다른 파벌 간의 갈등

**경제 차원**:
- 무엇을 교역하나? 왜?
- 화폐 시스템
- 마법이 경제에 어떤 영향?
- 자원 희소성

**사회 차원**:
- 계급 구조
- 종족 간 관계
- 마법 사용자의 지위
- 교육과 지식 전파

**문화 차원**:
- 종교와 신앙
- 예술과 오락
- 축제와 전통
- 가치관과 금기

### 역사 깊이

**최소 3세대 역사**:

**현대** (이야기 발생 시):
- 현 상태는?
- 주요 갈등과 문제
- 주요 세력과 인물

**부모 세대** (30-50년 전):
- 어떤 사건이 현대를 형성했나?
- 구세대가 기억하는 것
- 남은 문제들

**조부모 세대** (60-100년 전):
- 전설과 이야기
- 실전된 지식
- 역사적 트라우마

**고대** (더 오래전):
- 신화와 전설
- 문명의 흥망
- 현대 설정의 씨앗

## 종족과 생물 설계

### 지적 종족

**단일 특성 피하기**:
❌ 모든 엘프가 우아하고 고귀
❌ 모든 드워프가 탐욕스럽고 성질 급함
❌ 모든 오크가 야만적이고 호전적

**깊이 만들기**:
✅ 내부 다양성 (다른 문화, 가치관)
✅ 개인차 (성격이 제각각)
✅ 역사 복잡성 (좋은 역사와 나쁜 역사)

**종족 특성에 이유가 있어야**:
- 생리적 특징이 생존에 어떻게 도움이 되나?
- 문화적 특성은 어디서 왔나?
- 다른 종족과의 관계 역사

### 마법 생물

**설계 원칙**:
1. **생태적 지위**
   - 생태계에서 어떤 역할?
   - 무엇을 먹나? 무엇에게 먹히나?
   - 어떻게 번식하나?

2. **마법 원천**
   - 왜 마법이 있나?
   - 마법이 생존에 어떻게 도움?
   - 대가나 제한은?

3. **인간과의 관계**
   - 위험한가 유용한가?
   - 길들일 수 있나?
   - 인간은 어떻게 대응하나?

## 흔한 함정

### ❌ 과잉 설명

**문제**: 세계 설정 설명에 장을 써서, 이야기 추진이 멈춤

**해결**:
- 행동 속에서 설정을 보여주기
- 캐릭터가 알 필요가 있는 것만 설명
- 독자가 스스로 조합하게
- 정보가 플롯이나 캐릭터에 봉사해야

### ❌ 일관성 없는 규칙

**문제**: 마법/세계 규칙이 플롯 편의를 위해 바뀜

**해결**:
- 모든 주요 규칙을 미리 확립
- 확립된 규칙을 추적
- 예외에는 초기 암시가 필요
- 캐릭터가 제한 내에서 창의적으로 문제 해결하게

### ❌ 유럽 중세 기본 설정

**문제**: 모든 판타지가 유럽 중세의 변형

**해결**:
- 다른 문화와 시대 탐색
- 여러 문화 요소 혼합
- 독특한 사회 구조 창조
- 마법이 사회 발전을 어떻게 바꿀지 고려

### ❌ 선택받은 자 함정

**문제**: 주인공이 행동이 아닌 예언/혈통으로 특별함

**해결**:
- 주인공이 선택으로 특별해지게
- 예언이 있더라도 노력하게
- 함정을 전복하거나 해체
- 재능이 아닌 캐릭터 성장에 집중

## 판타지 서브장르

### 하이 판타지 (에픽 판타지)
- 완전히 가상의 세계
- 선과 악의 대립
- 서사시적 규모와 위험
- 마법이 세계의 일부

### 로우 판타지
- 현실 세계 + 마법 요소
- 더 작고 개인적인 위험
- 마법이 희귀하고 신비
- 더 현실적인 톤

### 어반 판타지
- 현대 도시 설정
- 숨겨진 마법 세계
- 보통 탐정/미스터리 요소
- 두 세계의 충돌

### 다크 판타지
- 도덕적 회색
- 공포와 폭력 요소
- 안티히어로 주인공
- 더 가혹한 결과

## Novel-Writer 명령과의 통합

### `/specify` 실행 시
- 핵심 세계 요소 정의 (마법, 종족, 지리)
- 판타지 서브장르 식별
- 확립해야 할 주요 규칙 나열
- 정보 공개의 리듬 계획

### `/plan` 중
- 세계 요소가 플롯에 미치는 영향 매핑
- 마법 시스템 전시 계획
- 다른 문화의 충돌 설계
- 세계 규칙의 일관성 확보

### `/write` 시
- 행동 속에서 세계를 보여주기, 쏟아붓지 않기
- 캐릭터가 세계 요소에 반응하게
- 감각 디테일로 세계를 생생하게
- 확립된 규칙의 일관성 유지

### `/analyze` 중
- 세계 규칙의 일관성 검사
- 마법 시스템의 논리 검증
- 모든 세계 요소에 목적이 있는지 확인
- 설정이 이야기에 봉사하는지 확인

## 세계관 구축 체크리스트

- [ ] 마법 시스템에 명확한 규칙과 제한
- [ ] 지리와 기후가 문화와 플롯에 영향
- [ ] 최소 3세대 역사 깊이
- [ ] 다수의 정치 세력에 명확한 동기
- [ ] 경제와 교역에 논리적 기반
- [ ] 종족/문화에 내부 다양성
- [ ] 마법 생물에 생태적 지위와 목적
- [ ] 종교/신앙 시스템에 영향력
- [ ] 사회 구조에 명확한 계급과 이동성
- [ ] 모든 규칙이 전체 이야기에서 일관

## 정보 공개 전략

### 빙산 원칙

**만든 것 vs 보여주는 것**:
- 100%를 만들되, 10-20%만 보여주기
- 독자가 모든 것을 알 필요 없음
- 깊이가 자신감과 일관성을 만듦
- 작가는 보여주는 것보다 더 많이 앎

### 공개 시기

**첫 번째 장**:
- 기본 세계관 (상세 설명이 아닌)
- 눈길을 끄는 마법/판타지 요소 하나
- 주인공의 세계 속 위치

**첫 25%**:
- 핵심 마법 시스템 규칙
- 주요 종족/문화
- 중심 갈등과 세계적 뿌리

**중반**:
- 확립된 요소 심화
- 세계 정치 복잡화
- 역사의 영향 공개

**후반**:
- 모든 세계 실마리 연결
- 깊은 역사 공개
- 세계 요소가 갈등을 어떻게 해결하는지 보여주기

## 독자 기대

**판타지 독자가 원하는 것**:
- 몰입적이고 믿을 수 있는 세계
- 일관되고 흥미로운 마법 시스템
- 복잡한 문화와 정치
- 전형적 함정에 대한 신선한 시각
- 이야기보다 더 큰 세계의 느낌

**판타지 독자를 좌절시키는 것**:
- 일관성 없는 세계 규칙
- 플롯에 편리한 마법
- 얕은 유럽 중세 모방
- 깊이 없는 세계
- 유기적 공개가 아닌 정보 쏟아붓기

---

**기억하세요**: 훌륭한 세계관 구축은 이야기의 기반이지 목적이 아닙니다. 세계는 캐릭터와 플롯에 봉사해야 하며, 캐릭터의 행동은 세계 규칙에 의해 형성되고 제한되어야 합니다. 깊이와 서사 유려함의 균형이 핵심입니다.
